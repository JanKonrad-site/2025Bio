7.5.1 Vibe coding: tvorba webové stránky s ChatGPT

V této kapitole se podíváš na tvorbu webové stránky z jiného pohledu: nebudeš jen ručně psát HTML a CSS, ale naučíš se využít ChatGPT jako pomocníka při návrhu, generování, kontrole a úpravě kódu.

Co se naučíš?

  • Vysvětlit, co je jazykový model – pochopíš, že ChatGPT nečte myšlenky, ale odpovídá podle zadaného promptu.
  • Rozpoznat slabý a silný prompt – zjistíš, proč nestačí napsat „vytvoř mi stránku“.
  • Navrhnout vlastní zadání pro web – naučíš se popsat účel, obsah, vzhled a technická omezení stránky.
  • Kontrolovat a upravovat AI výstup – využiješ znalosti HTML a CSS k tomu, abys poznal chyby a dokázal stránku vylepšit.
Tip: AI ti může pomoct psát kód rychleji, ale dobrý výsledek vznikne až tehdy, když víš, co chceš vytvořit, a dokážeš výstup zkontrolovat.
Otevřít ChatGPT Doporučení k promptům
1) Proč nestačí napsat „vytvoř mi stránku“?

Prompt je zadání, které posíláš jazykovému modelu. Když je zadání příliš obecné, model neví, jaký má být účel stránky, pro koho je určená, co má obsahovat ani jak má vypadat.

Slabý prompt: 

Vytvoř mi webovou stránku.

Tento prompt je problémový, protože neříká:

  • o čem má stránka být,
  • pro koho je určená,
  • jaké části má obsahovat,
  • jak má vypadat,
  • jaké HTML a CSS prvky má použít,
  • co nemá používat.
2) Co je jazykový model a proč potřebuje přesné zadání?

Jazykový model je systém, který pracuje s textem. Dostane vstup, kterému říkáme prompt, a podle něj vytvoří odpověď. U tvorby webu může odpovědí být návrh struktury stránky, HTML kód, CSS styly nebo vysvětlení chyby.

2.1 ChatGPT není čtenář myšlenek

ChatGPT neví automaticky, jakou stránku máš v hlavě. Když mu nedáš dostatek informací, doplní si chybějící části podle pravděpodobnosti. Výsledek pak může být funkční, ale nemusí odpovídat tomu, co jsi skutečně chtěl.

  • Čím přesnější zadání, tím použitelnější výsledek.
  • Čím méně kontextu, tím více si model domýšlí.
  • Člověk musí výsledek vždy zkontrolovat.
2.2 Proč jsme se učili HTML a CSS?

Základy HTML a CSS nejsou zbytečné ani ve chvíli, kdy používáš AI. Právě díky nim dokážeš poznat, jestli je výstup správně strukturovaný, čitelný a přiměřený tvým znalostem.

  • Rozumíš značkám jako h1, p, section, ul nebo a.
  • Dokážeš upravit barvy, mezery, zarovnání a velikosti.
  • Poznáš, když AI použije zbytečně složitý nebo nevhodný kód.
3) Jak vytvořit dobrý prompt pro webovou stránku?

Dobrý prompt má jasně popsat, co chceš vytvořit. Nejde jen o větu „udělej stránku“, ale o stručný návrh projektu.

  1. Urči téma stránky.
  2. Popiš účel stránky.
  3. Napiš, pro koho stránka je.
  4. Vyjmenuj požadované části stránky.
  5. Popiš vzhled a atmosféru.
  6. Uveď technická omezení.
  7. Požádej o vysvětlení kódu.
Pomůcka: Dobrý prompt funguje jako zadání pro spolužáka. Kdyby podle něj měl někdo jiný vytvořit stránku, měl by přesně vědět, co po něm chceš.
4) Praktická aktivita: vytvoř vlastní web s pomocí ChatGPT

Tvým úkolem je vytvořit jednoduchou jednostránkovou webovou stránku. Nejdříve si ji navrhneš, potom připravíš prompt, použiješ ChatGPT a nakonec výsledek zkontroluješ a upravíš.

  1. Vyber si téma stránky.
  2. Vyplň návrhový list: účel, cílová skupina, obsah, vzhled, omezení.
  3. Vytvoř konkrétní prompt podle šablony.
  4. Vlož prompt do ChatGPT.
  5. Zkopíruj HTML kód do editoru.
  6. Spusť stránku v prohlížeči.
  7. Zkontroluj výsledek pomocí checklistu.
  8. Uprav prompt nebo kód podle toho, co je potřeba zlepšit.

Možná témata:

  • školní klub,
  • oblíbený sport,
  • fiktivní kavárna,
  • třídní výlet,
  • osobní portfolio,
  • pozvánka na školní akci,
  • informační stránka o bezpečnosti na internetu.
5) Jak zkontrolovat a vylepšit výsledek?

První odpověď od ChatGPT nemusí být finální verze. Vibe coding znamená, že s modelem vedeš dialog: zadáš požadavek, zkontroluješ výsledek, upřesníš zadání a postupně stránku vylepšuješ.

5.1 Checklist kontroly
  • Má stránka správný hlavní nadpis?
  • Jsou použity přehledné sekce?
  • Je text srozumitelný pro cílovou skupinu?
  • Je HTML kód správně odsazený?
  • Je CSS přehledné a není zbytečně složité?
  • Funguje stránka v prohlížeči?
  • Vypadá stránka dobře i v užším okně?
  • Rozumíš hlavním částem kódu?
  • Umíš alespoň tři věci v kódu sám upravit?
5.2 Navazující prompty pro úpravy

Když výsledek není ideální, neptej se znovu úplně od začátku. Požádej o konkrétní úpravu.

  • Zjednoduš tento kód tak, aby používal jen základní HTML a CSS.
  • Vysvětli mi tento kód po částech jako pro začátečníka.
  • Zkontroluj, jestli je tento kód přehledný a správně strukturovaný.
  • Uprav vzhled stránky tak, aby působila moderněji, ale nepoužívej JavaScript.
  • Uprav CSS tak, aby stránka dobře vypadala i na mobilu.
6) Odevzdání a závěrečná reflexe

Na konci práce odevzdáš nejen hotovou stránku, ale také prompt a krátkou reflexi. Cílem není jen mít pěkný web, ale umět vysvětlit, jak vznikl.

  1. Odevzdej finální HTML soubor.
  2. Odevzdej finální prompt, který vedl k nejlepší verzi stránky.
  3. Napiš krátkou reflexi: co udělala AI, co jsi rozhodl ty a co jsi musel upravit.
  • Co mi ChatGPT výrazně usnadnil?
  • Co za mě ChatGPT nerozhodl?
  • Kde jsem musel použít znalosti HTML a CSS?
  • Jaký prompt fungoval lépe než první pokus?
  • Co bych příště zadal přesněji?

Shrnutí

  • Umíš vysvětlit, proč obecný prompt obvykle nestačí k vytvoření dobré stránky.
  • Dokážeš připravit konkrétní prompt pro tvorbu webu pomocí ChatGPT.
  • Víš, proč jsou znalosti HTML a CSS důležité i při práci s AI.
  • Umíš výsledek zkontrolovat, upravit a obhájit vlastní rozhodnutí.
Hlavní myšlenka: AI může napsat kód rychle, ale člověk musí vědět, co chce vytvořit, umět výsledek zkontrolovat a dokázat ho upravit.

Ukázka slabého a silného promptu

Slabý prompt je příliš obecný: 

Vytvoř mi webovou stránku.

Silnější prompt dává modelu konkrétní kontext: 

Vytvoř jednoduchou jednostránkovou HTML stránku pro žáka 7. ročníku gymnázia o školním filmovém klubu.

Stránka má obsahovat:
- nadpis,
- krátký úvod,
- tři sekce: Program, Jak se přihlásit, Pravidla klubu,
- jednoduché CSS přímo v souboru,
- světlý moderní vzhled,
- responzivní rozvržení,
- žádný JavaScript,
- komentáře v kódu, které vysvětlí hlavní části.

Použij pouze HTML a CSS, které jsme se učili: h1, h2, p, ul, li, a, section, class, margin, padding, color, background-color, border-radius, font-family.

Rozdíl je v tom, že druhý prompt určuje téma, cílovou skupinu, strukturu, vzhled i technická omezení.

Vyzkoušet v ChatGPT

Co je jazykový model?

Jazykový model je nástroj, který pracuje s textem. Na základě zadaného promptu vytváří odpověď, například vysvětlení, návrh textu nebo programový kód.

  • Nereaguje na naše myšlenky, ale na konkrétní zadání.
  • Když chybí důležité informace, model si je může domyslet.
  • Výsledek je potřeba kontrolovat, testovat a upravovat.
  • Znalost HTML a CSS pomáhá poznat, jestli je kód vhodný.

Při tvorbě webu proto nestačí jen požádat o stránku. Je potřeba popsat účel, obsah, vzhled, omezení a očekávaný výstup.

Šablona promptu pro tvorbu webu

Tuto šablonu můžeš upravit podle vlastního tématu: 

Chci vytvořit webovou stránku o: [téma]

Účel stránky je: [informovat / pozvat / představit projekt / propagovat akci / ukázat portfolio]

Cílová skupina je: [spolužáci / rodiče / učitelé / veřejnost]

Stránka má obsahovat tyto části:
1. [část]
2. [část]
3. [část]

Vzhled stránky:
- barvy:
- styl:
- rozvržení:
- atmosféra:

Technická omezení:
- použij pouze HTML a CSS,
- nepoužívej JavaScript,
- CSS vlož do tagu style v hlavičce,
- kód napiš jako jeden kompletní HTML soubor.

Požadavky na kód:
- používej sémantické prvky,
- přidej komentáře,
- kód má být přehledně odsazený,
- nepoužívej věci, které jsme se neučili, nebo je vysvětli.

Na konci mi napiš krátké vysvětlení, jak stránka funguje.
