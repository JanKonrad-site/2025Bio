Teorie: Co je HTML a jak se vyvíjelo

HTML (HyperText Markup Language) je značkovací jazyk, který popisuje obsah a strukturu webových stránek. Hypertext znamená text s odkazy na jiné dokumenty. Markup znamená „značkování“ obsahu pomocí tagů.

HTML vzniklo na přelomu 80. a 90. let ve švýcarském CERNu, kde Tim Berners-Lee vytvořil první verzi webu. Od té doby se HTML vyvíjelo až do dnešní podoby – tzv. HTML5, které podporuje sémantické značky, multimédia i moderní přístupnost.

  • 1989–1991: první návrh webu a HTML (Tim Berners-Lee).
  • 1995–1999: standardizace, verze 2.0–4.0, přidání CSS pro vzhled.
  • 2008–2014: HTML5 – nové značky, audio, video, formuláře.
  • Dnes: HTML jako „živý standard“, základ každé webové stránky.
Ukázkový postup

1. Otevři ve VS Code novou složku lekce-3-1-0 a vytvoř v ní soubor index.html.

2. Do souboru napiš pouze znak ! a stiskni Tab nebo Enter. VS Code automaticky vygeneruje HTML5 kostru:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>

</body>
</html>

3. Do <body> vlož první odstavec <p> s textem: „Proč je pro mě Internet důležitý?“

Úkoly k textu

  • 1) Otevři VS Code a vytvoř složku lekce-3-1-0.
  • 2) Přidej soubor index.html a použij zkratku ! → Tab k vygenerování základní struktury.
  • 3) Do <body> vlož <p> a napiš krátký odstavec o tom, proč je pro tebe Internet důležitý.
Detailní vysvětlení struktury
Zobrazit vzor
<!DOCTYPE html>

Tento řádek říká prohlížeči, že se jedná o dokument v jazyce HTML5. Bez něj by mohl prohlížeč spustit tzv. quirks mode (nepředvídatelné chování).

<html> ... </html>

Kořenový element celé stránky. Všechen kód HTML musí být uzavřen uvnitř tohoto tagu. Atribut lang="cs" nebo lang="en" udává jazyk stránky.

<head> ... </head>

Sekce pro metadata a nastavení stránky. Patří sem název stránky, odkazy na CSS, informace pro vyhledávače, favicon apod.

<meta>

Meta tagy poskytují informace o stránce. charset="UTF-8" určuje znakovou sadu (umožňuje zobrazit českou diakritiku). viewport pomáhá s responzivitou na mobilech.

<title> ... </title>

Název stránky, který se zobrazuje v záložce prohlížeče a používají ho i vyhledávače. Text mezi <title> a </title> je důležitý pro SEO.

<body> ... </body>

Hlavní část stránky – zde je veškerý obsah, který uvidí uživatel (texty, obrázky, odkazy, formuláře…). To je místo, kam budeme psát HTML kód.
