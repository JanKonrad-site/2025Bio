3.1.6 První kroky s CSS – jak vdechnout webu barvy

Co se naučíš?

  • Co je CSS (Cascading Style Sheets) a proč ho web potřebuje.
  • Jak propojit HTML stránku s CSS souborem.
  • Jak funguje selektor, vlastnost a hodnota.
  • Rozdíl mezi type, class a ID selektorem.
  • Jak použít barvy (pojmenované, RGB, HEX) a vnořené prvky (div).

1️⃣ Co je CSS

CSS neboli Cascading Style Sheets znamená kaskádové styly. Pomocí CSS měníme vzhled prvků, které jsme vytvořili v HTML – barvy, písmo, okraje, rozložení, pozadí nebo animace.

HTML je obsah: co na stránce je. CSS je vzhled: jak to vypadá. Společně tvoří to, čemu říkáme webová prezentace.

🧩 Krok 1 – Vytvoření struktury stránky

Otevři Visual Studio Code, vytvoř složku moje_stranka a v ní dva soubory: index.html a style.css.

Do souboru index.html napiš následující kód:

<!DOCTYPE html>
<html lang="cs">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Moje první CSS</title>
  <link rel="stylesheet" href="./style.css">
</head>
<body>

  <h1>Ahoj světe!</h1>
  <p>Tohle je moje první stylování v CSS.</p>

</body>
</html>

Ulož, otevři stránku v prohlížeči – zatím vypadá obyčejně. Teď přidáme CSS.

2️⃣ Jak CSS funguje

CSS používá jednoduchou logiku: vyber prvek (selektor) → napiš, jak má vypadat. Každé pravidlo se skládá z:

  • selektoru – např. p nebo h1,
  • vlastnosti – např. color, font-size,
  • hodnoty – např. red, 20px.

Ukázka:

p {
  color: green;
  font-size: 18px;
}

Tímto nastavíme všem odstavcům zelenou barvu textu a velikost písma 18px.

🧩 Krok 3 – Napiš první styly

Do souboru style.css napiš:

body {
  background-color: rgb(240, 240, 255);
  color: #333333;
  font-family: Arial, sans-serif;
}

h1 {
  color: darkblue;
  text-align: center;
}

p {
  color: darkgreen;
}

Ulož a aktualizuj stránku. Vidíš barevné texty a světlé pozadí? Gratuluju – to je tvůj první CSS výsledek!

3️⃣ Práce s kontejnery (div)

Element <div> je univerzální kontejner – můžeme do něj vložit jiný obsah, seskupit ho a nastavit mu styl. Používá se k oddělení částí stránky (např. sekce, boxy, panely).

Přidej do HTML tento kód:

<div>
  <h2>Div je kontejner, do kterého patří obsah.</h2>
  <p>Zde je obyčejný odstavec.</p>
</div>

A do CSS přidej stylování pro všechny div:

div {
  background-color: rgb(240, 240, 255);
  color: #333333;
  border-radius: 10px;
  padding: 20px;
  margin: 20px;
}

Všimni si: každý div teď má zaoblené rohy, vnitřní odsazení a světle modré pozadí. padding je vnitřní mezera, margin je vnější mezera.

🧩 Krok 4 – Třídy a ID selektory

Někdy potřebujeme, aby některé prvky měly jiný vzhled než ostatní. K tomu slouží třídy (class) a ID. 

<p class="info">Toto je info text</p>
<p id="warning">Toto je varování</p>
<p>Obyčejný odstavec</p>

A teď do CSS přidej:

.info { color: blue; }
#warning { color: red; font-weight: bold; }

.info označuje všechny prvky se třídou info (tečka značí třídu). #warning označuje prvek s ID warning (mřížka značí ID). ID používáme jen jednou na stránce, třídu můžeme použít vícekrát.

5️⃣ Kaskáda a pořadí stylů

Když má jeden prvek více stylů, rozhoduje kaskáda (pořadí a přesnost). Například tento styl: 

p:last-child {
  color: purple;
}

Znamená: poslední odstavec v daném kontejneru (např. uvnitř <div>) bude mít fialovou barvu. Selektor :last-child vybírá „poslední dítě“ – to už je tzv. pseudo-třída.

Pokud má jeden odstavec třídu .info (modrou) a zároveň je poslední v divu (fialovou), prohlížeč vyhodnotí, který styl má vyšší prioritu. V tomto případě p:last-child – tedy fialová.

💡 Výsledná stránka by měla vypadat takto:
<!DOCTYPE html>
<html lang="cs">
<head>
  <link rel="stylesheet" href="./style.css">
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Moje první CSS</title>
</head>
<style>
  p:last-child { color: purple; }
  div {
    background-color: rgb(240, 240, 255);
    color: #333333;
    border-radius: 10px;
    padding: 20px;
    margin: 20px;
  }
  .info { color: blue; }
  #warning { color: red; font-weight: bold; }
</style>
<body>

  <h1>Ahoj světe!</h1>
  <p>Tohle je moje první stylování v CSS</p>

  <div>
    <h2>Div je kontejner, místo kam můžeme přiřadit obsah.</h2>
    <p>Zde je obyčejný odstavec.</p>
  </div>

  <div>
    <p class="info">Toto je info text</p>
    <p id="warning">Toto je varování</p>
    <p>Obyčejný odstavec</p>
  </div>

</body>
</html>
Cvičení 3.2.2 – Vytvoř základ osobní karty

Teď už z jednoduché stránky uděláš skutečnou osobní kartu. Procvičíš si selektory, barvy, div, class a ID.

Tvůj cíl

Přeměnit obyčejný text na první verzi osobní karty s několika odlišnými částmi.

Kroky
  1. Navazuj na předchozí projekt a pracuj ve stejné složce.
  2. Vytvoř hlavní obsahový box, který bude fungovat jako karta.
  3. Do karty vlož nadpis, krátký popis a alespoň dva další textové prvky.
  4. Jeden text označ jako informační – ten bude stylovaný pomocí class.
  5. Jeden text označ jako důležitý nebo varovný – ten bude stylovaný pomocí ID.
  6. Nastav stránce základní barvy a písmo.
  7. Vyzkoušej si rozdíl mezi stylováním podle názvu tagu, třídy a ID.
  8. Uprav vzhled tak, aby karta působila jako jeden celek, ne jako nahodilé texty pod sebou.
Na co si dát pozor
  • ID používej jen pro jeden konkrétní prvek.
  • Class je vhodná pro styl, který bys mohl později použít vícekrát.
  • Texty si napiš vlastní – může jít o skutečné portfolio, herní profil nebo profil fiktivní osoby.
Hotovo budeš mít tehdy, když:
  • máš na stránce jednu jasně oddělenou kartu,
  • používáš alespoň jeden div, jednu class a jedno id,
  • je na první pohled vidět rozdíl mezi běžným, informačním a důležitým textem.

📘 Shrnutí lekce

  • CSS určuje vzhled stránky – barvy, písmo, rozložení.
  • Pravidlo = selektor + vlastnosti + hodnoty.
  • Třída (.nazev) může být víckrát, ID (#nazev) jen jednou.
  • Kontejner <div> seskupuje obsah a snadno se styluje.
  • Kaskáda = pořadí a přesnost stylů (poslední a konkrétnější pravidla mají přednost).
