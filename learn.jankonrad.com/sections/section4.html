4. Úvod do programování v Pythonu

Co se naučíš?

V této kapitole pochopíš, co je to programování obecně, jak počítač vykonává příkazy, co je to skript a jak Python čte kód. Ukážeme si i první příkazy v Pythonu jako print() a input(). Naučíš se také, kde Python získat, jak ho nainstalovat a kde psát kód.

  • Co je programování: myšlení o problému + instrukce pro počítač
  • Jak počítač čte kód: řádek po řádku, shora dolů
  • Skript: posloupnost příkazů
  • Program: větší celek složený z více skriptů či modulů
  • Příkazy print() a input(): první praktické ukázky
  • IDE a instalace: kde získat Python a psát kód
Tip: Program je jako scénář – provádí se řádek po řádku odshora dolů.
🖥️ Co je IDE a kde psát kód

IDE (Integrated Development Environment) je prostředí, ve kterém můžeš psát, ukládat a spouštět kód. Obsahuje zvýraznění syntaxe, konzoli, ladicí nástroje atd.

  • Visual Studio Code (VS Code) – populární, zdarma, přehledný editor
  • Thonny – jednoduché IDE speciálně pro začátečníky v Pythonu
  • Online editory – např. Replit.com, PythonAnywhere.com, náš školní editor

💡 Doporučujeme VS Code pro pokročilejší uživatele, nebo Thonny pro začátek.

📥 Kde stáhnout Python
  • Oficiální stránka: https://www.python.org/downloads/
  • Vyber si verzi pro svůj operační systém (Windows, macOS, Linux)
  • Spusť instalaci a zaškrtni Add Python to PATH (důležité pro příkazovou řádku)
▶️ Jak spustit Python skript
  • 1. V IDE (např. Thonny, VS Code) – klikni na tlačítko Run
  • 2. Příkazová řádka (Windows): python muj_program.py
  • 3. Terminál na MacBooku: python3 muj_program.py
📌 Tip: Pro testování můžeš spustit python nebo python3 a zkusit příkazy přímo.
📜 Jak počítač čte kód v Pythonu

Python čte skript odshora dolů, vykonává příkazy jeden po druhém. Říkáme tomu sekvenční vykonávání.

print("A")
print("B")
print("C")

Tento skript vypíše: A, B, C – vždy v tomto pořadí.

📁 Skript a program
  • Skript: jednoduchý soubor obsahující posloupnost příkazů
  • Program: rozsáhlejší celek – může se skládat z více skriptů a knihoven
🖨️ Funkce print()

Funkce print() vypíše text do konzole. Používáme ji pro komunikaci s uživatelem a pro výstup informací.

print("Hello World")
⌨️ Funkce input()

Funkce input() zastaví program a čeká, až uživatel něco napíše a stiskne Enter.

input("Stiskni Enter pro ukončení")

Používá se např. na konci skriptu, aby se program nezavřel okamžitě.

Úkoly

  • 1. Otevři editor níže
  • 2. Napiš print("Hello, World!")
  • 3. Napiš print("Tvé jméno")
  • 4. Přidej input("Stiskni Enter pro ukončení")
  • 5. Spusť a sleduj pořadí vykonání

🔑 Shrnutí

  • 📌 Python čte kód odshora dolů, řádek po řádku
  • 📌 Skript je soubor s příkazy, program může být větší celek
  • 📌 print() vypisuje text, input() čeká na vstup
  • 📌 Spouštět můžeš v IDE, v příkazové řádce nebo v terminálu
Tip: Každý velký program začíná malým „Hello, World!“
