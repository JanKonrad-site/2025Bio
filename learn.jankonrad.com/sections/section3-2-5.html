Pokročilé CSS – pozadí, průhlednost, přechody, stíny, kombinátory

Co se naučíš?

  • Nastavit background image a ovlivnit jeho chování (repeat, size, attachment).
  • Použít průhlednost přes rgba() a pochopit rozdíl oproti opacity.
  • Vytvářet linear a radial gradient + kombinovat gradient s obrázkem.
  • Přidat stíny přes box-shadow (offset, blur, spread, inset).
  • Používat kombinátory selektorů (child, descendant, next sibling, subsequent sibling).

3.2.1 (1/5) Background image – obrázek jako pozadí stránky

Obrázek můžeš použít jako pozadí – typicky pro celou stránku (např. na body). V CSS se k tomu používá background-image s funkcí url().

Pozadí – co ovládáme nejčastěji
  • background-image: url("...") – vybere soubor obrázku (relativní cesta v projektu).
  • background-repeat – když je obrázek menší, prohlížeč ho opakovaně „dlaždí“ (výchozí). Pro jednu kopii dáš no-repeat.
  • background-size – určuje škálování obrázku (nejčastěji contain nebo cover).
  • background-attachment – chování při scrollu: scroll (běžné) nebo fixed (obrázek zůstává „přilepený“ na místě).
Tip pro zadání: obrázek dej do stejné složky jako index.html, pak stačí url("nazev-obrazku.jpg").
Contain vs Cover

background-size: contain;

  • Celý obrázek je vidět, nic se neusekne.
  • Může vzniknout „prázdné místo“, když se poměr stran obrázku neshoduje s oknem.

background-size: cover;

  • Obrázek pokryje celé pozadí (bez prázdných míst).
  • Některé části se mohou uříznout, aby se zachoval poměr stran.

Prakticky (krok za krokem)

V index.html měj základ a do <body> vlož:

<section class="demo-wrap">
  <h1>Background image</h1>
  <p>Tady bude obsah – uděláme trochu textu, aby šlo scrollovat.</p>

  <div class="scroll-block">
    <p>Scroll test…</p>
    <p>Scroll test…</p>
    <p>Scroll test…</p>
    <p>Scroll test…</p>
    <p>Scroll test…</p>
    <p>Scroll test…</p>
  </div>
</section>
Asset: do stejné složky ulož obrázek (např. brandenburger.jpg). Název si uprav podle toho, co student stáhne. 

body{
  margin: 0;
  padding: 0;
  font-family: system-ui, sans-serif;

  background-image: url("brandenburger.jpg");
  background-repeat: no-repeat;
  background-size: cover;
  background-attachment: fixed;
}

.demo-wrap{ padding: 16px; }

.scroll-block{
  background: rgba(255,255,255,.8);
  padding: 16px;
  border-radius: 12px;
  max-width: 720px;
}

Zkus scroll: když je background-attachment: fixed, pozadí zůstává na místě i při posunu stránky.

3.2.2 (2/5) Průhlednost: rgba() vs opacity

Průhlednost můžeš řešit dvěma způsoby: (A) průhledná barva přes rgba() nebo (B) průhlednost celého prvku přes opacity. Rozdíl je zásadní.

rgba(r,g,b,a)

rgba() je jako rgb(), ale má navíc čtvrtý parametr a (alpha). Alpha je číslo 0 až 1.

  • 1 = plně neprůhledné
  • 0 = úplně průhledné
  • 0.7 = částečně průhledné (jde „vidět skrz“)
opacity

opacity funguje podobně jako alpha (0 až 1), ale ovlivní celý prvek – pozadí, border i text uvnitř.

Když dáš opacity: 0.7 na box s textem, zneprůhlední se i text. Když chceš průhledné jen pozadí a text nechat „plný“, použij rgba() na background-color. 

<section class="demo-wrap">
  <h1>rgba vs opacity</h1>

  <div class="alpha">Pozadí přes rgba() – text zůstane „normální“</div>
  <div class="opacity">Opacity na celý prvek – průhledný i text</div>
</section>

.alpha, .opacity{
  padding: 16px;
  margin: 12px 0;
  border: 2px solid rgba(0,0,0,.25);
  border-radius: 12px;
  max-width: 720px;
}

.alpha{
  background-color: rgba(255,255,255,.7);
}

.opacity{
  background-color: white;
  opacity: .7;
}

3.2.3 (3/5) Gradienty: linear + radial + kombinace s obrázkem

Gradienty jsou v CSS „obrázky“, proto se nastavují přes background-image. Dva typy, které teď použijeme: linear-gradient a radial-gradient.

Linear gradient

Základ: background-image: linear-gradient(red, yellow); (plynulý přechod mezi barvami).

Směr může být první parametr, např. to right, to bottom right, nebo úhel 50deg.

Procenta jsou „zastávky“ v rozsahu 0–100 %. Např. black 30% znamená: černá drží do 30 %, pak se teprve začne měnit.

Radial gradient

Základ: background-image: radial-gradient(red, yellow); (přechod začíná uprostřed a „vyzařuje“ ven).

U radial gradientu je 0 % uprostřed a 100 % na okrajích. Proto red 20% znamená: červená drží v centru do 20 %. 

<section class="demo-wrap">
  <h1>Gradienty</h1>

  <div class="g linear">linear-gradient</div>
  <div class="g radial">radial-gradient</div>

  <h2>Black hole (gradient + image)</h2>
  <div class="g black-hole"></div>
</section>
Asset: ulož obrázek pro black-hole (např. space.jpg) do stejné složky. 

.g{
  width: 200px;
  height: 200px;
  border: 1px solid black;
  display: grid;
  place-items: center;
  margin-bottom: 16px;
}

.linear{
  background-image: linear-gradient(to right, red, yellow);
}

.radial{
  background-image: radial-gradient(red 20%, yellow);
}

Gradient musí být „nad“ obrázkem → proto jde radial-gradient první, až pak url(). 

.black-hole{
  width: 300px;
  height: 300px;
  border: 1px solid black;

  background-image:
    radial-gradient(
      black 30%,
      white 31%,
      transparent 40%
    ),
    url("space.jpg");

  background-size: cover;
}

3.2.4 (4/5) Stíny: box-shadow

Stín dělá „hloubku“. Nastavuje se přes box-shadow. První dva parametry jsou povinné – posun na ose X a Y.

box-shadow – význam hodnot
  • offset-x (povinné) – kladné doprava, záporné doleva
  • offset-y (povinné) – kladné dolů, záporné nahoru
  • blur-radius (volitelné) – rozmazání (vyšší = měkčí stín)
  • spread-radius (volitelné) – zvětšení / zmenšení stínu
  • color (volitelné) – barva stínu (často průhledná)
Inset a kombinace více stínů
  • inset dá stín „dovnitř“ – jako vyhloubení.
  • Více stínů oddělíš čárkou – můžeš mít normální i inset současně.
box-shadow:
  0 8px 18px rgba(0,0,0,.25),
  inset 0 0 0 2px rgba(255,255,255,.25);

<section class="demo-wrap">
  <h1>Box-shadow</h1>
  <div class="shadow-card">Karta se stínem</div>
</section>

.shadow-card{
  width: 320px;
  padding: 16px;
  border-radius: 12px;
  background: white;

  box-shadow:
    20px 10px 12px rgba(0,0,0,.25),
    inset 0 0 0 2px rgba(0,0,0,.08);
}

3.2.5 (5/5) Kombinátory selektorů – vztahy mezi prvky

Kombinátory vybírají prvky podle vztahů v HTML struktuře. Typicky: rodič → dítě, nebo sourozenci (prvky se stejným rodičem).

Kombinátory
  • Child (přímé dítě): header > h2
  • Descendant (potomek kdekoliv uvnitř): div h2
  • Next sibling (nejbližší sourozenec hned za): h2 + p
  • Subsequent sibling (všichni sourozenci za): h2 ~ p
Specifičnost selektorů

Když dva CSS zápisy míří na stejný prvek, prohlížeč vybere ten „přesnější“ (specifičtější). 

article > p { ... }
div > article > p { ... }

Druhý selektor je konkrétnější (vede přes víc kroků), takže má vyšší specifičnost a obvykle „vyhraje“. 

<section class="demo-wrap">

  <header class="container">
    <h2>Nadpis v headeru</h2>
    <p>První odstavec v headeru</p>
    <p>Druhý odstavec v headeru</p>
  </header>

  <div class="container">
    <h2>Nadpis v divu</h2>
    <article>
      <p>Odstavec v article (uvnitř divu)</p>
    </article>
  </div>

</section>

.container{
  border: 2px solid rgba(0,0,0,.15);
  padding: 12px;
  margin-bottom: 14px;
  border-radius: 12px;
  background: rgba(255,255,255,.75);
}

/* child: přímé dítě (jen h2, které je přímo v header.container) */
header.container > h2{
  color: darkslateblue;
}

/* descendant: všechny h2 někde uvnitř divu (i kdyby byly hlouběji) */
div.container h2{
  color: darkgreen;
}

/* next sibling: jen první p hned za h2 (v headeru) */
header.container h2 + p{
  text-decoration: underline;
}

/* subsequent sibling: všechny p za h2 (v headeru) */
header.container h2 ~ p{
  opacity: .9;
}
Cvičení 3.2.5 – Finální vzhled mini portfolia

V této kapitole dáš svému mini portfoliu finální atmosféru. Využiješ background image, rgba, gradient, box-shadow a jednodušší práci s výběrem prvků podle jejich vztahů.

Tvůj cíl

Proměnit dosavadní kartu v hotovou úvodní sekci mini portfolia s výrazným vizuálním stylem.

Kroky
  1. Připrav si obrázek, který bude fungovat jako pozadí stránky nebo hlavní sekce.
  2. Nastav obrázek tak, aby nepůsobil jako malá dlaždice opakovaná po celé stránce.
  3. Rozhodni se, jestli chceš, aby pozadí při scrollování zůstávalo na místě, nebo se posouvalo.
  4. Nad pozadí umísti hlavní panel portfolia.
  5. Pomocí průhledné barvy vytvoř jemné překrytí, aby byl text dobře čitelný.
  6. Vyzkoušej rozdíl mezi průhledným pozadím a průhledností celého prvku.
  7. Přidej gradient – může být lineární nebo radiální podle atmosféry, kterou chceš vytvořit.
  8. Přidej stín, aby hlavní karta vystoupila od pozadí.
  9. Pokud máš v portfoliu více textových prvků vedle sebe nebo pod sebou, zkus některé vybrat pomocí vztahových selektorů.
  10. Uprav výsledný vzhled tak, aby stránka působila jako hotová úvodní sekce osobního mini webu.
Na co si dát pozor
  • Pozadí má doplňovat obsah, ne ho rušit.
  • Text musí být i po přidání obrázku a gradientu dobře čitelný.
  • Stíny a efekty používej s mírou.
  • Když použiješ opacity, zkontroluj, jestli nezmizela i čitelnost textu.
Hotovo budeš mít tehdy, když:
  • portfolio má vlastní atmosféru a vizuální styl,
  • obsahuje pozadí nebo výraznější hero sekci,
  • hlavní karta je zvýrazněná pomocí overlaye, gradientu nebo stínu,
  • výsledek působí jako hotový malý osobní web.

📘 Shrnutí

  • Pozadí: background-image + repeat, size, attachment.
  • Průhlednost: rgba() = průhledná barva, opacity = průhledný celý prvek včetně textu.
  • Gradienty: linear-gradient + radial-gradient, procenta jsou zastávky 0–100 %.
  • Gradient + obrázek: více „background image“ oddělíš čárkou, pořadí určuje vrstvu (první je nahoře).
  • Stíny: box-shadow (x/y offset povinné), blur, spread, inset, více stínů čárkou.
  • Kombinátory: >, mezera, +, ~ vybírají prvky podle vztahů v HTML.
