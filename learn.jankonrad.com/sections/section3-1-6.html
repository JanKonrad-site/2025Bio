3.1.5 Úvod do stylování CSS

Co se naučíš?

  • Rozpoznat tři typy CSS stylů – inline, internal, external.
  • Pochopit prioritu (kaskádu) mezi jednotlivými styly.
  • Vytvořit jednoduchý dokument s propojením HTML a CSS.

Teorie: Jak funguje stylování v CSS

CSS (Cascading Style Sheets – kaskádové styly) určují vzhled HTML prvků. Každý styl se dá přidat třemi způsoby:

  • Inline styl – přímo v tagu pomocí atributu style="".
  • Internal styl – v rámci stránky mezi značkami <style></style>.
  • External styl – v samostatném souboru .css.

Tyto styly se na sebe vrství (kaskádují) – pokud se stejný prvek styluje vícekrát, platí ten nejblíže definovaný nebo nejvíce specifický.

Jak funguje kaskáda

Kaskáda znamená, že styly se aplikují podle pořadí a úrovně specifity:

  • 1. Výchozí styl prohlížeče
  • 2. External stylesheet (soubor .css)
  • 3. Internal styl v <style>
  • 4. Inline styl v atributu style=""

Inline má vždy nejvyšší prioritu, ale je méně přehledný.

Ukázkový projekt z hodiny

V této lekci jste vytvořili dva soubory: index.html a style.css.

Struktura HTML dokumentu s ukázkami všech tří způsobů stylování:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>

    <!-- INTERNAL STYLING -->
    <style>
        h1, h2, h3 {
            color: salmon;
            text-align: center;
        }
    </style>

    <!-- EXTERNAL STYLESHEET -->
    <link rel="stylesheet" href="./style.css">
</head>
<body>

    <h1>Stylování CSS</h1>

    <!-- INLINE STYL -->
    <p style="color: lightgoldenrodyellow;">Inline stylování</p>

    <!-- INTERNAL STYL -->
    <h1>Nadpis první úrovně</h1>
    <h2>Nadpis druhé úrovně</h2>
    <h3>Nadpis třetí úrovně</h3>

    <!-- EXTERNAL STYL -->
    <p>První odstavec</p>
    <p>Druhý odstavec</p>
    <p>Třetí odstavec</p>
    <p>Čtvrtý odstavec</p>

</body>
</html>
Všimni si:
  • Inline styl (style="...") platí jen pro jeden prvek.
  • Internal styl (v <style>) ovlivní všechny nadpisy.
  • External styl (soubor .css) styluje všechny odstavce <p>.

p {
    color: lightpink;
    font-size: 20px;
    text-align: center;
}

body {
    background-color: lightgray;
}

Tento soubor ovlivňuje všechny odstavce a barvu pozadí celé stránky.

Jak to funguje:
<link rel="stylesheet" href="./style.css"> propojí tvůj CSS soubor s HTML.

  • Inline styl – má nejvyšší prioritu. Zde barví jeden odstavec na světle žlutou.
  • Internal styl – barví všechny nadpisy (H1–H3) na lososovou a centruje je.
  • External styl – barví všechny odstavce <p> na růžovo a přidává pozadí celé stránce.

Výsledkem je stránka s šedým pozadím, lososovými nadpisy, růžovými odstavci a jedním světle žlutým textem, který má inline styl.

Shrnutí lekce

  • Inline styl = nejvyšší priorita, ale nepřehledný.
  • Internal styl = přehledný pro jednu stránku.
  • External styl = profesionální řešení pro celý web.

CSS soubory udržuj přehledné, formátuj automaticky (Shift+Alt+F), a používej komentáře (/* ... */) pro popis sekcí.
