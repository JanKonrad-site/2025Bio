2.2.3 Kvadratické funkce a rovnice

Co se naučíš?

  • Rozlišit kvadratickou funkci (y = ax² + bx + c) a kvadratickou rovnici (ax²+bx+c=0).
  • Pochopit význam parametrů a, b, c – otevření, posun osy, posun nahoru/dolů.
  • Najít vrchol paraboly (u=-b/(2a), v=f(u)) a průsečíky s osami.
  • Umět řešit kvadratickou rovnici pomocí diskriminantu a vysvětlit počet řešení.
  • Prakticky vytvořit tabulku hodnot a XY graf v Excelu, dopočítat vrchol a kořeny.
Tip: V Excelu použij XY (bodový) graf. Hledání řešení (Goal Seek) najde X, pro které Y=0, tedy řešení rovnice.

1. Kvadratická funkce

Obecný tvar kvadratické funkce je:

y = ax² + bx + c

  • a: směr a šířka paraboly (a>0 nahoru, a<0 dolů).
  • b: posouvá osu souměrnosti (a tím vrchol) po ose X.
  • c: posouvá graf nahoru/dolů (průsečík s osou Y je [0; c]).

Vrchol paraboly: u = -b/(2a), v = f(u).

Intuice: a = otevření a šířka, b = vodorovný posun vrcholu, c = svislý posun.

2. Kvadratická rovnice

Když hledáme, kdy je funkce rovna nule, dostáváme rovnici:

ax² + bx + c = 0

Řešení = průsečíky paraboly s osou X.

Diskriminant

D = b² − 4ac

  • D > 0 → dva reálné kořeny.
  • D = 0 → jeden dvojnásobný kořen (dotyk v bodě vrcholu).
  • D < 0 → žádné reálné kořeny.
Vzorce pro kořeny

x₁ = (-b - √D) / (2a), x₂ = (-b + √D) / (2a)

3. Výpočet v Excelu

  1. Parametry: D1:a=1, E1:b=2, F1:c=-3.
  2. Do A napiš X (-5 až 5). Do B vzorec: =$D$1*A2^2 + $E$1*A2 + $F$1.
  3. Diskriminant G1: =E1^2 - 4*D1*F1 → 16.
  4. Kořeny:
    • G2: =(-E1 - ODMOCNINA(G1))/(2*D1) → -3
    • G3: =(-E1 + ODMOCNINA(G1))/(2*D1) → 1
  5. Vrchol:
    • G4 (u): =-E1/(2*D1) → -1
    • G5 (v): =D1*G4^2 + E1*G4 + F1 → -4
  6. Označ A:B a vlož XY graf s hladkou čarou. Přidej do grafu body pro kořeny a vrchol.
V Excelu lze řešení najít i přes Data → Analýza hypotéz → Hledání řešení.

4. Konkrétní příklad z našeho sešitu

Naše parametry: a=1, b=2, c=-3

Funkce: y = x² + 2x − 3

  • Vrchol: [−1; −4]
  • Diskriminant: 16
  • Kořeny: x₁ = −3, x₂ = 1
  • Průsečík s osou Y: [0; −3]

5. Úkoly k procvičení

  • 1) Vytvoř tabulku a graf pro a=1, b=0, c=-6. Najdi vrchol a kořeny.
  • 2) Ověř, že pro a=1, b=2, c=-3 vychází kořeny −3 a 1.
  • 3) Najdi příklad, kdy D<0. Jak vypadá parabola?
  • 4) Použij „Hledání řešení“ a nech Excel najít X, kdy Y=0.

Shrnutí celé hodiny

  • Kvadratická funkce má tvar y = ax² + bx + c.
  • Kvadratická rovnice ax² + bx + c = 0 určuje průsečíky s osou X.
  • Diskriminant D=b²−4ac určuje počet řešení.
  • Vrchol paraboly: u=-b/(2a), v=f(u).
  • Excel umožňuje spočítat tabulku, graf, vrchol i kořeny přímo pomocí vzorců.
Tip: Pamatuj si: funkce popisuje celý graf, rovnice hledá jen konkrétní body (řešení).

4.1 Hudební příklad – Buben (kvadratická aproximace první fáze pohybu)

Cíl: Na jednoduchém modelu ukázat, jak kvadratická funkce popíše první „oblouk“ výchylky bubnové membrány po úderu a jak z ní v Excelu automaticky najdeme vrchol (maximum) a kořeny (návrat do klidu).

Model a jednotky
  • Proměnná x: čas po úderu v milisekundách [ms].
  • Proměnná y: relativní výchylka membrány (bezrozměrně, např. v „jednotkách amplitudy“).
  • Funkce: y = ax^2 + bx + c se zvolenými parametry:
    • a = −0,25 (jednotky ≈ 1/ms²) – udává „rychlost návratu“ (zakřivení dolů).
    • b = 2 (jednotky ≈ 1/ms) – udává „sílu/tempo růstu“ výchylky po úderu.
    • c = 0 – počáteční výchylka je nulová (y(0)=0).
    Tedy konkrétně: y = −0,25·x^2 + 2·x + 0.
Poznámka: Skutečný pohyb je tlumená sinusovka. Kvadratika zde slouží jako jednoduchá aproximace první fáze (od úderu po první návrat).
Krok 1: Připrav tabulku v Excelu
  1. Do D1:F1 napiš záhlaví: a, b, c. Do D2:F2 hodnoty: -0,25, 2, 0.
  2. Do A1 napiš čas [ms], do B1 výchylka y.
  3. Do A2 napiš 0, do A3 0,5. Obě buňky označ a táhni úchopem dolů, až dojdeš k 8,0 (krok 0,5 ms dává hezký hladký graf).
  4. Do B2 napiš vzorec a zkopíruj dolů:
    = $D$2*A2^2 + $E$2*A2 + $F$2
    (Dolar $ „uzamkne“ buňky s parametry.)
Krok 2: Vrchol (maximum) a kořeny automaticky

Mimo tabulku (třeba do oblasti H:G) si připrav pomocné výpočty:

  • Diskriminant D (např. do H2 napiš „D“ a do I2 vzorec):
    = E2^2 - 4*$D$2*F2
    Pro naše parametry vyjde D = 4.
  • Kořeny (návrat do klidu) – do H4 „x₁“, I4:
    =(-E2 - ODMOCNINA(I2)) / (2*$D$2)
    do H5 „x₂“, I5:
    =(-E2 + ODMOCNINA(I2)) / (2*$D$2)
    Vyjde x₁ = 0 ms a x₂ = 8 ms.
  • Vrchol paraboly – do H7 „u (čas)“, I7:
    = -E2 / (2*$D$2)
    a do H8 „v (výchylka)“, I8:
    = $D$2*I7^2 + E2*I7 + F2
    Vyjde u = 4 ms a v = 4 (vrchol [4; 4]).
Krok 3: Graf (XY – bodový se spojnicí)
  1. Označ sloupce A:B a vlož Vložit → Bodový (XY) → Hladká spojnica.
  2. Do grafu přidej samostatné body:
    • Vrchol: Pravým → Vybrat data → Přidat sérii, X-hodnota = buňka I7, Y-hodnota = I8.
    • Kořeny: Přidej novou sérii, X-hodnoty = I4:I5, Y-hodnoty zadej ručně jako {0,0} (nebo dvě buňky s nulou).
    Pak v Formátování nastav těmto sériím jen „značky“ (bez čáry) a případně popisky dat („x=0“, „x=8“, „vrchol“).
Volitelně: Kořen bez vzorce (Hledání řešení)
  1. Do K2 napiš „x*“, do L2 třeba 1 (výchozí odhad). Do K3 „y(x*)“, do L3:
    = $D$2*L2^2 + $E$2*L2 + $F$2
  2. Data → Analýza hypotéz → Hledání řešení: Nastav Buňku pro nastavení = L3, na hodnotu = 0, změnou buňky = L2.
  3. Spusť – najde jeden kořen. Když do L2 dáš jiný odhad (např. 7), najde ten druhý.
Hotové výsledky (kontrola)
  • Kořeny: x = 0 ms a x = 8 ms (membrána je v klidu na začátku a po 8 ms).
  • Vrchol: v čase 4 ms dosáhne výchylka maxima y = 4.
Co dělají parametry? a (záporné) stáčí oblouk dolů a udává „rychlost návratu“, b posouvá čas vrcholu (větší b ⇒ vrchol později a výš), c je počáteční výchylka.
Shrnutí pro tebe (učitelsky) Výběr parametrů a jednotek je teď jasně daný: x [ms], y [rel. výchylka], a [1/ms²], b [1/ms], c [1]. 2.2.3 Kvadratické funkce a rovnice – koncert (praktický model)

Co se naučíš?

  • Pochopit model ceny lístku, poptávky diváků a tržby jako kvadratickou funkci.
  • Umět určit vrchol paraboly (optimální cenu) a kořeny (hranice, kdy je tržba nulová).
  • Spočítat vše automaticky v Excelu a sestrojit XY graf.
Idea: Vyšší cena → menší návštěvnost. Tržba = cena × počet diváků. Výsledek je kvadratická funkce s maximem.

1) Model koncertu: cena → poptávka → tržba

Uvažujme jednoduchou (lineární) poptávku po lístku:

D(p) = n0 − k·p

  • p … cena lístku (v €).
  • D(p) … počet diváků (osob).
  • n0 … maximální počet diváků, když je lístek „zdarma“.
  • k … o kolik osob klesne zájem při zdražení o 1 €.

Tržba (výnos) je: R(p) = p · D(p) = p(n0 − k·p) = −k·p² + n0·p

Tedy kvadratická funkce:

R(p) = a·p² + b·p + c, kde a = −k (záporné), b = n0, c = 0.

Význam parametrů: a (záporné) určuje „jak rychle“ po překročení optimální ceny tržba padá; b je „startovní růst“ (čím větší potenciální publikum, tím strmější nárůst); c=0, protože při ceně 0 € je tržba 0 €.

2) Konkrétní příklad (realistická čísla)

  • n0 = 1000 diváků (kdyby to bylo zdarma).
  • k = 20 (při zdražení o 1 € ubyde cca 20 lidí).

Tržba: R(p) = −20·p² + 1000·p

  • a = −20, b = 1000, c = 0.
  • Vrchol (optimální cena): p* = −b / (2a) = −1000 / (2·(−20)) = 25 €.
  • Maximální tržba: R(25) = −20·25² + 1000·25 = 12 500 €.
  • Kořeny (nulová tržba): p = 0 € (zdarma) a p = b/|a| = 1000/20 = 50 € (moc drahé, nikdo nepřijde).

3) Výpočet v Excelu (automaticky)

Krok A — Parametry a tabulka
  1. Do D1:F1 napiš záhlaví: a, b, c. Do D2:F2 napiš: -20, 1000, 0.
  2. Do A1 napiš cena p [€], do B1 tržba R(p) [€].
  3. Do A2 zadej 0, do A3 1. Obě označ a táhni dolů až do 50 (krok 1 €).
  4. Do B2 vlož vzorec a zkopíruj dolů:
    = $D$2*A2^2 + $E$2*A2 + $F$2
    (Dolar $ uzamkne parametry v řádku 2.)
Krok B — Vrchol a kořeny z parametrů

Do volných buněk (třeba H:I) si udělej pomocné výpočty:

  • Diskriminant D = b² − 4ac — do H2 „D“, do I2:
    = E2^2 - 4*$D$2*F2
    Zde vyjde D = 1 000 000.
  • Kořeny:
    x1 (H4), I4: =(-E2 - ODMOCNINA(I2)) / (2*$D$2)
    x2 (H5), I5: =(-E2 + ODMOCNINA(I2)) / (2*$D$2)
    Vyjde 0 a 50.
  • Vrchol:
    p* (H7), I7: = -E2 / (2*$D$2)
    R(p*) (H8), I8: = $D$2*I7^2 + E2*I7 + F2
    Vyjde 25 a 12 500.
Krok C — Najdi optimum „bez vzorce“ (Hledání řešení)
  1. Do K2 napiš „p* (odhad)“, do L2 dej např. 10.
  2. Do K3 „R(p*)“, do L3:
    = $D$2*L2^2 + $E$2*L2 + $F$2
  3. Otevři Data → Analýza hypotéz → Hledání řešení: nastav Buňku pro nastavení = L3, na maximum, změnou buňky = L2. Excel najde p* ≈ 25.
Krok D — Graf (XY bodový se spojnicí)
  1. Označ A:B a vlož Vložit → Bodový (XY) → Hladká spojnice.
  2. Přidej body pro kořeny a vrchol jako samostatné série (X = I4:I5, Y = nuly; a X = I7, Y = I8). Nastav viditelné značky a popisky.

4) Propojení s kvadratickou funkcí

Máme R(p) = a·p² + b·p + c se a < 0. Vrchol je:

p* = −b/(2a), R(p*) = a·p*² + b·p* + c

Kořeny (kde je tržba nulová):

p = 0 a p = −b/a = n0 / k

Interpretace: a je záporné – parabola klesá po obou stranách vrcholu; b reprezentuje „potenciální publikum“; c=0, protože při 0 € nemáme tržbu.

5) Úkoly k procvičení

  • 1) Vypočti a v grafu vyznač kořeny a vrchol pro R(p) = −20p² + 1000p.
  • 2) Změň parametry na n₀ = 800, k = 16 (tedy R(p) = −16p² + 800p) a znovu najdi optimální cenu. Co se změnilo a proč?
  • 3) (Volitelné) Přidej fixní náklady 5 000 € a zjisti, při jaké ceně je zisk maximální. (Tip: nejdřív definuj Z(p) = R(p) − 5000.)
  • 4) Najdi optimální cenu pomocí Hledání řešení bez použití vzorce pro vrchol.

Shrnutí

  • Tržba z koncertu: R(p) = −k·p² + n₀·p je kvadratická funkce s maximem.
  • Optimální cena: p* = −b/(2a); kořeny: p = 0 a p = n₀/k.
  • Excel spočítá tabulku, kořeny, vrchol i maximum automaticky; graf (XY) vizualizuje celou situaci.
