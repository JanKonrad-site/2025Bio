2.2.3 Kvadratické funkce a rovnice – celá hodina (podle listu „3_Hodina“)

Co se naučíš?

  • Rozlišit kvadratickou funkci (y = ax² + bx + c) a kvadratickou rovnici (ax²+bx+c=0).
  • Poznat typy kvadratických rovnic (úplné / neúplné) a znát podmínku a ≠ 0.
  • Řešit kvadratické rovnice pomocí diskriminantu a odvodit počet řešení.
  • V Excelu sestavit tabulku X–Y, spočítat vrchol, diskriminant, kořeny a vytvořit XY graf.
  • Na konci mít hotové 2 příklady s grafem: (1) y = x² + 2x − 3, (2) y = x² − 6.
Tipy:
  • Pro graf používej XY (bodový) s hladkou čarou, ne spojnicový.
  • F4 přepíná relativní/absolutní adresování (A1 → $A$1).
  • Dnešní datum: Ctrl+; nebo =DNES().

1) Teorie: funkce, rovnice, typy a diskriminant

Kvadratická funkce: y = ax² + bx + c, kde a ≠ 0. Grafem je parabola.

  • a: směr a „šířka“ paraboly (a>0 nahoru; a<0 dolů).
  • b: ovlivňuje polohu osy souměrnosti (vrchol: u=-b/(2a)).
  • c: svislý posun (průsečík s osou Y je [0; c]).

Kvadratická rovnice: ax² + bx + c = 0.

Typy rovnic:

  • Úplná – a, b, c jsou nenulové (typicky).
  • Neúplná – chybí b (např. ax² + c = 0) nebo chybí c (např. ax² + bx = 0).
  • Podmínka: vždy a ≠ 0 (jinak je to lineární rovnice / funkce).
Diskriminant a počet řešení

D = b² − 4ac

  • D > 0 – dva reálné kořeny x₁, x₂ = (-b ± √D)/(2a).
  • D = 0 – jeden dvojnásobný kořen x = -b/(2a) (v bodě vrcholu).
  • D < 0 – žádné reálné řešení (graf neprotíná osu X).

2) Příklad 1: y = x² + 2x − 3

Naše parametry: a=1, b=2, c=-3 ⇒ funkce y = x² + 2x − 3.

  • Vrchol: u=-b/(2a) = -1, v=f(u) = -4 ⇒ V[−1; −4].
  • Diskriminant: D = 2² − 4·1·(−3) = 16.
  • Kořeny rovnice: x₁ = -3, x₂ = 1.
  • Průsečík s osou Y: [0; c] = [0; −3].
Excel – tabulka, výpočty, graf
  1. Do buněk napiš parametry: D4:a=1, E4:b=2, F4:c=-3.
  2. Sloupec A: X od -5 do 5 (po 1). Sloupec B (od B4): =$D$4*A4^2 + $E$4*A4 + $F$4 → kopíruj dolů.
  3. Diskriminant G1: =E1^2 - 4*D1*F1 (vyjde 16).
  4. Kořeny:
    • G2 (x₁): =(-E1 - ODMOCNINA(G1))/(2*D1) → -3
    • G3 (x₂): =(-E1 + ODMOCNINA(G1))/(2*D1) → 1
  5. Vrchol:
    • G4 (u): =-E1/(2*D1) → -1
    • G5 (v): =D1*G4^2 + E1*G4 + F1 → -4
  6. Označ A:B a vlož XY (bodový) s hladkou čarou. Přidej bod „Vrchol“ (X=G4, Y=G5) a body „Kořeny“ (X=G2, G3; Y=0, 0).
Rychlá kontrola: Hodnoty sedí např.: (−5;12), (−3;0), (−1;−4), (1;0), (5;32).

3) Příklad 2 – nový úkol: y = x² − 6

Parametry: a=1, b=0, c=-6 ⇒ funkce y = x² − 6.

  • Vrchol: u=0, v=-6 ⇒ V[0; −6].
  • Diskriminant: D = 0² − 4·1·(−6) = 24 (kladný).
  • Kořeny rovnice: x = ±√6 (≈ −2,449 a 2,449).
  • Průsečík s osou Y: [0; −6].
Excel – tabulka, výpočty, graf
  1. Parametry: D1:a=1, E1:b=0, F1:c=-6.
  2. Sloupec A: X od -5 do 5 (po 0,5 nebo 1). B2: =$D$1*A2^2 + $E$1*A2 + $F$1 → dolů.
  3. Diskriminant G1: =E1^2 - 4*D1*F1 → 24.
  4. Kořeny (pokud chceš číselně):
    • G2 (x₁): =(-E1 - ODMOCNINA(G1))/(2*D1) → ≈ -2,449
    • G3 (x₂): =(-E1 + ODMOCNINA(G1))/(2*D1) → ≈ 2,449
  5. Vrchol:
    • G4 (u): =-E1/(2*D1) → 0
    • G5 (v): =D1*G4^2 + E1*G4 + F1 → -6
  6. Vlož XY graf z A:B. Přidej body „Vrchol“ (G4,G5) a „Kořeny“ (G2,0) a (G3,0).
Rychlá kontrola: Sedět budou např.: (−5;19), (−3;3), (−2;−2), (0;−6), (2;−2), (3;3), (5;19).

4) Procvičení a nástroje

  • A) Doplň tabulku a graf pro libovolné a, b, c. Zapiš D, vrchol a kořeny.
  • B) Najdi příklad s D<0 (např. a=1, b=0, c=5). Vysvětli, proč nejsou reálné kořeny a jak vypadá graf.
  • C) Vyzkoušej Data → Analýza hypotéz → Hledání řešení (Goal Seek): nastav buňku s y na 0 a hledat x (dostaneš jeden z kořenů).
  • D) (Volitelné) Doplň do listu i vrcholový tvar: y = a(x-u)² + v, kde u=-b/(2a), v=f(u), a porovnej s původním tvarem.

    Shrnutí

    • Kvadratická funkce: y = ax² + bx + c; kvadratická rovnice: ax² + bx + c = 0.
    • Typy rovnic: úplná, neúplná; podmínka: a ≠ 0.
    • Diskriminant D=b²−4ac určuje počet reálných řešení.
    • Vrchol (u,v) s u=-b/(2a), v=f(u); průsečíky s osami zjišťuj dosazením.
    • Na konci máš hotové 2 příklady s grafem: y = x² + 2x − 3 a y = x² − 6.
    Tip: Při nesedících hodnotách zkontroluj, zda vzorec v B skutečně odkazuje na buňky s parametry (použij $D$1, $E$1, $F$1).
