2.2.3 Kvadratické funkce a rovnice

Co se naučíš?

  • Rozlišit kvadratickou funkci (y = ax² + bx + c) a kvadratickou rovnici (ax²+bx+c=0).
  • Pochopit význam parametrů a, b, c – otevření, posun osy, posun nahoru/dolů.
  • Najít vrchol paraboly (u=-b/(2a), v=f(u)) a průsečíky s osami.
  • Umět řešit kvadratickou rovnici pomocí diskriminantu a vysvětlit počet řešení.
  • Prakticky vytvořit tabulku hodnot a XY graf v Excelu, dopočítat vrchol a kořeny.
Tip: V Excelu použij XY (bodový) graf. Hledání řešení (Goal Seek) najde X, pro které Y=0, tedy řešení rovnice.

1. Kvadratická funkce

Obecný tvar kvadratické funkce je:

y = ax² + bx + c

  • a: směr a šířka paraboly (a>0 nahoru, a<0 dolů).
  • b: posouvá osu souměrnosti (a tím vrchol) po ose X.
  • c: posouvá graf nahoru/dolů (průsečík s osou Y je [0; c]).

Vrchol paraboly: u = -b/(2a), v = f(u).

Intuice: a = otevření a šířka, b = vodorovný posun vrcholu, c = svislý posun.

2. Kvadratická rovnice

Když hledáme, kdy je funkce rovna nule, dostáváme rovnici:

ax² + bx + c = 0

Řešení = průsečíky paraboly s osou X.

Diskriminant

D = b² − 4ac

  • D > 0 → dva reálné kořeny.
  • D = 0 → jeden dvojnásobný kořen (dotyk v bodě vrcholu).
  • D < 0 → žádné reálné kořeny.
Vzorce pro kořeny

x₁ = (-b - √D) / (2a), x₂ = (-b + √D) / (2a)

3. Výpočet v Excelu

  1. Parametry: D1:a=1, E1:b=2, F1:c=-3.
  2. Do A napiš X (-5 až 5). Do B vzorec: =$D$1*A2^2 + $E$1*A2 + $F$1.
  3. Diskriminant G1: =E1^2 - 4*D1*F1 → 16.
  4. Kořeny:
    • G2: =(-E1 - ODMOCNINA(G1))/(2*D1) → -3
    • G3: =(-E1 + ODMOCNINA(G1))/(2*D1) → 1
  5. Vrchol:
    • G4 (u): =-E1/(2*D1) → -1
    • G5 (v): =D1*G4^2 + E1*G4 + F1 → -4
  6. Označ A:B a vlož XY graf s hladkou čarou. Přidej do grafu body pro kořeny a vrchol.
V Excelu lze řešení najít i přes Data → Analýza hypotéz → Hledání řešení.

4. Konkrétní příklad z našeho sešitu

Naše parametry: a=1, b=2, c=-3

Funkce: y = x² + 2x − 3

  • Vrchol: [−1; −4]
  • Diskriminant: 16
  • Kořeny: x₁ = −3, x₂ = 1
  • Průsečík s osou Y: [0; −3]

5. Úkoly k procvičení

  • 1) Vytvoř tabulku a graf pro a=1, b=0, c=-6. Najdi vrchol a kořeny.
  • 2) Ověř, že pro a=1, b=2, c=-3 vychází kořeny −3 a 1.
  • 3) Najdi příklad, kdy D<0. Jak vypadá parabola?
  • 4) Použij „Hledání řešení“ a nech Excel najít X, kdy Y=0.

Shrnutí celé hodiny

  • Kvadratická funkce má tvar y = ax² + bx + c.
  • Kvadratická rovnice ax² + bx + c = 0 určuje průsečíky s osou X.
  • Diskriminant D=b²−4ac určuje počet řešení.
  • Vrchol paraboly: u=-b/(2a), v=f(u).
  • Excel umožňuje spočítat tabulku, graf, vrchol i kořeny přímo pomocí vzorců.
Tip: Pamatuj si: funkce popisuje celý graf, rovnice hledá jen konkrétní body (řešení).
