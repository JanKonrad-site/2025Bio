8.1.5 Edit Mode: úpravy tvarů (Vertex / Edge / Face, Extrude, Loop Cut)

Poprvé začneš měnit skutečný tvar objektu. Naučíš se přepínat výběr vrcholů, hran a ploch, používat extrude a přidávat dělení pomocí loop cut. Na konci si „jen pro zábavu“ vyrobíš monstrózní tvar.

Základní přepínače v Edit Mode

  • Tab – přepíná Object Mode / Edit Mode.
  • Výběr v Edit Mode: Vertex / Edge / Face (ikonky nahoře).
  • Klávesy 1/2/3 přepínají Vertex/Edge/Face (pokud nepřekáží jiné mapování kláves).
  • Transformace fungují stejně: G, R, S.
  • Zrušení akce: Esc nebo pravé tlačítko.
Tip: Pokud nevidíš levý panel nástrojů, přepíná se klávesou T.
1) Extrude (E) – nejdůležitější modelovací nástroj
  1. V novém souboru vyber default cube.
  2. Tab do Edit Mode → přepni na Face výběr.
  3. Vyber jednu plochu a stiskni E (Extrude) → tahej ven → potvrď LMB.
Dvě typické pasti:
  • Extrude dovnitř (vytvoří „vnitřní“ plochy a tenké stěny) – začátečníkům to často rozhodí tvar.
  • Extrude a zrušení: když dáš E a pak zrušíš pohyb, může vzniknout extruze „na místě“ (dvojité body/plošky). Když si nejsi jistý, vrať se o krok zpět přes Ctrl+Z.
2) Loop Cut (Ctrl+R) – přidání dělení pro přesnější extruze
  1. Přepni se na Edge/Face dle potřeby (není kritické, Ctrl+R funguje v Edit Mode).
  2. Stiskni Ctrl + R a najeď myší na objekt – objeví se smyčka.
  3. LMB potvrdí směr, druhé LMB určí polohu. Pro „rychlý střed“ použij double LMB.
  4. Alternativa: LMB → pokud omylem pohneš myší, dej RMB pro „snap“ do středu.
Cíl: Udělej loop cut „napůl“, přepni na Face a vyextruduj jen jednu část (procvičení přesnosti).
3) „Hraj si“: vytvoř monstrózní tvar (bez stresu)

Teď jde jen o to získat cit pro E a Ctrl+R. Vytvoř několik loop cutů a vyextruduj různé plochy ven – výsledek nemusí být hezký.

Pravidlo pro začátek: Vyhni se extruzi „těsně vedle stěny“, pokud si nejsi jistý, co se stane. U některých kombinací může vzniknout vnitřní plocha, která později dělá problémy.

Na konci tento objekt klidně smaž – cílem je jen procvičit ovládání.

4) Pozor: Shift+A v Edit Mode přidá geometrii do stejného objektu + Separace (P)

Když jsi v Edit Mode a dáš Shift + A, často uvidíš jen nabídku Mesh. Vše, co přidáš, se vloží do stejného objektu (jedné mesh).

  1. Pokud omylem přidáš „druhý objekt“ v Edit Mode, ve skutečnosti je to součást jedné mesh.
  2. Chceš-li to rozdělit: P → Separate → By Loose Parts.
Prakticky: Nezapamatuj si to nazpaměť – stačí vědět, proč se to děje a že to jde spravit přes P.

Shrnutí

  • Umíš přepnout do Edit Mode (Tab) a zvolit Vertex/Edge/Face.
  • Umíš extrudovat plochy (E) a přidat loop cut (Ctrl+R).
  • Víš o typických chybách (extruze „na místě“, extruze dovnitř).
  • Chápeš rozdíl mezi přidáním objektu v Object Mode vs v Edit Mode a umíš separovat přes P.
