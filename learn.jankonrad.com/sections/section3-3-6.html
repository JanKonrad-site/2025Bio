3.3.6 Smyčky a opakování – while, for, break a continue

Co se naučíš?

Smyčky jsou jeden z nejdůležitějších nástrojů v programování – umožní ti opakovat akci bez toho, aby ses u toho „kopíroval“ desetkrát stejný kód.

  • while: opakování „dokud platí podmínka“.
  • for: ideální na počítání (vše v jedné hlavičce).
  • infinite loop: proč vzniká a jak se mu vyhnout.
  • break: ukončení smyčky okamžitě.
  • continue: přeskočení jedné iterace.
Tip: Vždy si hlídej, že se podmínka smyčky jednou stane false (jinak vytvoříš nekonečnou smyčku).

1) while smyčka (opakuj, dokud platí podmínka)

while funguje podobně jako if – má podmínku v závorkách. Rozdíl je v tom, že neprovede blok jen jednou, ale opakuje ho pořád dokola, dokud je podmínka true.

Jak přesně probíhá jedna smyčka?
  1. Vyhodnotí se podmínka (true/false).
  2. Když je true, provede se blok v { }.
  3. Na konci bloku typicky aktualizuješ „počítadlo“.
  4. Vrátí se zpět na začátek a znovu kontroluje podmínku.
  5. Když se podmínka stane false, smyčka skončí.

Ukázka: počítání 0 až 9

let counter = 0;

while (counter < 10) {
  console.log(counter);
  counter++; // bez tohohle by to byla nekonečná smyčka
}
Pozor: Když nikdy nezměníš hodnotu, která ovlivňuje podmínku, smyčka může běžet „navždy“ (infinite loop).
2) Cvičení: while smyčka (1 až 10 + speciální zpráva pro 7)

Napiš while smyčku, která vypíše čísla od 1 do 10. Ale když bude hodnota 7, místo čísla vypiš text: "7 je moje oblíbené číslo".

Řešení (zobraz až po pokusu) 
let i = 1;

while (i <= 10) {
  let message = i;

  if (i === 7) {
    message = "7 je moje oblíbené číslo";
  }

  console.log(message);
  i++;
}
Proč to funguje: Smyčka opakuje blok, dokud i <= 10. Uvnitř vždycky zvýšíš i, takže podmínka jednou přestane platit.
3) for smyčka (vše na jednom řádku: start, podmínka, krok)

for je často nejlepší volba pro „počítací“ smyčky. V závorce jsou tři části oddělené středníky:

  • Inicializace (spustí se jednou): let i = 0
  • Podmínka (kontroluje se před každou iterací): i < 10
  • Krok (spustí se po každé iteraci): i++
for (let i = 0; i < 10; i++) {
  console.log(i);
}
Kdy co použít?
for → když víš „kolikrát“ (typicky počítání).
while → když nevíš přesný počet opakování a jedeš „dokud se něco nestane“.
4) break a continue (ovládání smyčky)

break smyčku okamžitě ukončí. continue přeskočí zbytek aktuální iterace a skočí na další.

break (ukonči smyčku)
for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i);
}
// vypíše: 0 1 2 3 4
continue (přeskoč iteraci)
for (let i = 0; i < 10; i++) {
  if (i === 5) continue;
  console.log(i);
}
// vypíše: 0 1 2 3 4 6 7 8 9
5) Cvičení: sudá čísla 2 až 100, ale 50 přeskoč

Napiš program, který počítá od 2 do 100 a vypisuje jen sudá čísla. Ale když je číslo 50, tak ho nevypíše (přeskočí).

Řešení A (podmínka „sudé a ne 50“) 
for (let i = 2; i <= 100; i++) {
  if (i % 2 === 0 && i !== 50) {
    console.log(i);
  }
}
Řešení B (continue na 50) 
for (let i = 2; i <= 100; i++) {
  if (i === 50) {
    continue;
  }
  if (i % 2 === 0) {
    console.log(i);
  }
}
Řešení C (krok po 2) 
for (let i = 2; i <= 100; i += 2) {
  if (i !== 50) console.log(i);
}

let i = 10;

while (i < 5) {

  i++;
}

Tahák: Smyčky – rychlé shrnutí

  • while: while (podmínka) { ... } → opakuje blok, dokud je podmínka true.
  • for: for (start; podmínka; krok) { ... } → start proběhne 1×, podmínka se kontroluje před každou iterací, krok se provede po iteraci.
  • Nekonečná smyčka: vznikne, když se podmínka nikdy nestane false (typicky zapomeneš i++).
  • break: okamžitě ukončí smyčku.
  • continue: přeskočí aktuální iteraci a pokračuje další.
  • Sudá/lichá čísla: n % 2 === 0 → sudé, n % 2 !== 0 → liché.
Nejčastější vzorec: „počítadlo + podmínka + krok“. Jakmile tohle držíš pod kontrolou, smyčky přestanou být strašák.
Nápověda: while 1–10 + speciální 7
  • Začni proměnnou i = 1.
  • Podmínka smyčky bude i <= 10.
  • Uvnitř smyčky použij if (i === 7) a vypiš speciální text.
  • Na konci nezapomeň i++, jinak vznikne nekonečná smyčka.
Nápověda: sudá čísla 2–100, ale ne 50
  • Sudé číslo poznáš přes i % 2 === 0.
  • Číslo 50 buď vyfiltruješ podmínkou i !== 50, nebo ho přeskočíš přes continue.
  • Nejkratší varianta: for (let i = 2; i <= 100; i += 2) a jen ošetřit 50.
Detail: co přesně dělá while

while = „dokud platí, opakuj“

  • Podmínka se kontroluje před každou iterací.
  • Když je hned na začátku false, smyčka se nespustí ani jednou.
  • Nejčastější chyba: zapomenout změnit proměnnou v podmínce → nekonečná smyčka.
let i = 0;
while (i < 3) {
  console.log(i);
  i++;
}
// 0 1 2
Detail: proč je výstup 1 2 3

Krokování:

  1. i = 1, test 1 < 4 → true → vypíše 1 → i++ → 2
  2. 2 < 4 → true → vypíše 2 → i++ → 3
  3. 3 < 4 → true → vypíše 3 → i++ → 4
  4. 4 < 4 → false → konec
Detail: continue v for smyčce

continue znamená „přeskoč zbytek iterace a jdi na další“. V tomhle kódu se přeskočí právě chvíle, kdy i === 3. 

for (let i = 1; i <= 5; i++) {
  if (i === 3) continue;
  console.log(i);
}
// 1 2 4 5
Detail: proč se smyčka nespustí

while testuje podmínku hned na začátku. Když je false, blok se nikdy neprovede. 

let i = 10;
while (i < 5) {
  console.log(i);
  i++;
}
// i < 5 je hned false, takže se nic nevypíše
