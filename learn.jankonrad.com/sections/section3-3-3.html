3.3.3 Porovnávací operátory a podmínky (if / else / else if)

Co se naučíš?

V této kapitole si vysvětlíme, jak v JavaScriptu porovnávat hodnoty (např. „je 5 větší než 7?“) a jak na základě výsledku rozhodovat pomocí if, else a else if.

  • Porovnávací operátory: >, <, >=, <=, ==, !=
  • Boolean výsledek: porovnání vždy vrací true nebo false
  • Striktní porovnání: proč je bezpečnější používat === a !==
  • Podmínky: jak funguje if, else, else if
  • Časté pasti: „divné“ výsledky kvůli automatické konverzi typů
Tip: V reálném kódu většinou používej === a !==. Vyhneš se překvapením z automatické konverze typů.
Porovnávací operátory (comparison operators)

Porovnávací operátory umožňují porovnat dvě hodnoty. Výsledek porovnání je vždy boolean – tedy true nebo false.

Operátor Význam Příklad Výsledek
> větší než 5 > 7 false
< menší než 5 < 7 true
>= větší nebo rovno 5 >= 5 true
<= menší nebo rovno 4 <= 5 true
== rovnost (může převádět typy) 5 == "5" true (pozor)
=== striktní rovnost (bez převodu typů) 5 === "5" false
!= nerovnost (může převádět typy) 0 != false false (pozor)
!== striktní nerovnost 0 !== false true
Pozor: JavaScript je „loosely typed“ – některé operátory jako == a != umí hodnoty automaticky převádět (coercion). To může vést k nečekaným výsledkům (např. false == 0 je true).
Podmínky – if, else, else if

Podmínky umožňují programu rozhodovat se. Když je podmínka pravdivá (true), vykoná se blok kódu v { }. Když je nepravdivá (false), blok se přeskočí.

Základní syntaxe
let age = 19;

if (age > 20) {
  console.log("Věk je větší než 20");
} else {
  console.log("Věk je 20 nebo méně");
}

else if použiješ, když chceš otestovat více možností (např. různé jazyky, různé úrovně věku apod.).

Cvičení: Kalkulačka slevy pro e-shop

Představ si, že programuješ pokladnu e-shopu. Pokud je nákup za 100 $ nebo více, zákazník dostane 30% slevu.

Zadání

  • Vytvoř proměnnou totalPrice (celková cena nákupu).
  • Zkontroluj, jestli je totalPrice dost na slevu (≥ 100).
  • Pokud ano: spočítej cenu po slevě (30% dolů) a vypiš ji do konzole.
  • Pokud ne: spočítej, kolik chybí do 100, a vypiš hlášku typu: „Utrať ještě 15 $ a získáš 30% slevu!“
let totalPrice = 85; // změň hodnotu a testuj různé případy

if (totalPrice >= 100) {
  let discountedPrice = totalPrice * 0.7; // 30% sleva = platíš 70%
  console.log(discountedPrice);
} else {
  let amountNeeded = 100 - totalPrice;
  console.log(`Utrať ještě $${amountNeeded} a získáš 30% slevu!`);
}

Shrnutí kapitoly

  • Porovnání vždy vrací true nebo false (boolean).
  • Základní operátory: >, <, >=, <=, ==, !=.
  • Bezpečnější varianta: používej === a !== (hlídají i datový typ).
  • if: kód se spustí jen když je podmínka true.
  • else: spustí se, když je podmínka v if false.
  • else if: když máš více možností, testuješ je postupně – jakmile jedna vyjde, zbytek se neřeší.
Tip: Když chceš „více větví“ rozhodování, je else if přehlednější a efektivnější než několik samostatných if pod sebou.
