3.1 Databáze – úvod (Excel vs. databáze, dotazy, SQLite)

Základní orientace: co je databáze, v čem se liší od Excelu, co znamenají nejčastější zkratky a jak budeme databázi prakticky používat v Pythonu pomocí SQLite a jednoduchých SQL dotazů.

Co se v této kapitole naučíš?

  • Co je databáze – uložená data, se kterými se dá systematicky pracovat.
  • Excel vs. databáze – kdy stačí tabulka a kdy se vyplatí databázový systém.
  • Dotaz (query) – jak se databáze „ptá“ a co vrací jako výsledek.
  • SQL – základní jazyk dotazů (v této fázi jen CREATE/INSERT/SELECT).
  • SQLite – databáze v jednom souboru .db, kterou budeme používat v Pythonu.
  • Zkratky – DB, DBMS, SQL, CRUD (v minimálním rozsahu).
Tip: Teorie bude jen tolik, kolik potřebujeme pro praktické programování v Pythonu.
Primární akce (např. zadání / pracovní list) Python sqlite3 (dokumentace)
1) Co je databáze a co je DBMS

Databáze (DB) jsou data uložená tak, aby se v nich dalo spolehlivě vyhledávat a pracovat s nimi. DBMS (Database Management System) je program, který databázi spravuje – přijímá dotazy, ukládá změny a vrací výsledky.

  • DB = data (obsah).
  • DBMS = software, který s daty pracuje (správce databáze).
2) Excel vs. databáze (kdy co použít)

Excel (tabulkový procesor) je výborný pro výpočty, přehledy a menší tabulky. Databáze se hodí ve chvíli, kdy data potřebuješ ukládat dlouhodobě, přehledně a spolehlivě pro aplikaci (např. v Pythonu).

  • Excel – rychlá práce s tabulkou, filtrování, grafy, výpočty.
  • Databáze – systematické ukládání dat + dotazy + možnost rozšiřovat aplikaci nad daty.
Praktická pomůcka: Jakmile už data „nejsou jen tabulka“, ale stávají se základem programu (aplikace), databáze dává smysl.
3) Jak budeme pracovat v tomto kurzu (SQLite + Python)

Budeme používat SQLite. Je to databáze, která je uložená v jednom souboru (např. movies.db). Python se k ní připojí a posílá jí SQL dotazy.

  • SQLite = databáze v souboru .db (bez instalace serveru).
  • Python = náš „klient“ (program), který databázi ovládá.
  • SQL = jazyk, kterým se databáze ptáme a měníme data.

Schéma komunikace:

  1. Python se připojí k databázi (connect).
  2. Pošle dotaz (execute).
  3. Databáze vrátí výsledek (např. seznam řádků) nebo potvrdí změnu.
4) Co je dotaz (query) a co je SQL

Dotaz je instrukce pro databázi – například „vytvoř tabulku“, „ulož záznam“ nebo „najdi všechny filmy“. V relačních databázích se dotazy typicky zapisují v SQL.

Nejčastější dotazy, které budeme v začátku používat:

  • CREATE TABLE – vytvoří tabulku (strukturu).
  • INSERT – vloží nový řádek (záznam).
  • SELECT – načte data (výsledek dotazu).

Mini ukázka SQL (jen pro představu):

CREATE TABLE movies (id INTEGER PRIMARY KEY, title TEXT);
INSERT INTO movies (title) VALUES ('Matrix');
SELECT COUNT(*) FROM movies;
Důležité: SQL příkazy budeme psát a zkoušet postupně v dalších lekcích, vždy s ověřením výsledku.
5) Zkratky (minimum pro start)
  • DB – Database (databáze, data).
  • DBMS – Database Management System (program, který DB spravuje).
  • SQL – Structured Query Language (jazyk dotazů).
  • Query – dotaz (SQL příkaz).
  • CRUD – Create/Read/Update/Delete (4 základní operace nad daty; zatím použijeme hlavně Create + Read).
✅ Cvičení (krátké)
  1. Napiš vlastními slovy rozdíl: Excel vs. databáze (2–3 věty).
  2. Vysvětli rozdíl: DB vs. DBMS (1 věta).
  3. Napiš 3 příklady „dotazů“, které by mohla databáze dostat (česky, bez SQL). Např. „přidej nový film“, „spočítej filmy“.
  4. Urči, co budeme v kurzu používat: SQLite (1 věta: proč je vhodná pro začátek).

Shrnutí

  • Excel je tabulkový nástroj; databáze je základ pro aplikace pracující s daty a dotazy.
  • DB jsou data, DBMS je software, který data spravuje a odpovídá na dotazy.
  • Dotaz (query) je instrukce pro databázi – typicky v SQL.
  • Budeme používat SQLite: databázi v souboru .db ovládanou z Pythonu.
