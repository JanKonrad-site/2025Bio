1.1 Retuš fotografie: ořez, narovnání a odstranění vad

Otevřeš fotografii, narovnáš ji a ořízneš bez rušivých okrajů. Potom odstraníš praskliny a skvrny pomocí nástrojů pro retuš (Spot Healing, Healing Brush, Clone Stamp, Patch). Na konci vše správně uložíš a vyexportuješ.

Co se naučíš?

  • Otevření souboru – přetažením do PhotoPea nebo přes File → Open.
  • Ořez a narovnání – Crop Tool (C), rotace, potvrzení, volba „Delete cropped pixels“.
  • Bezpečná práce – ukládat průběžně + duplikace vrstvy jako záloha.
  • Retuš vad – Spot Healing Brush, Healing Brush, Clone Stamp, Patch Tool.
  • Export – JPG s kvalitou 100 % a správným názvem souboru.
Tip: PhotoPea může být občas pomalejší. Dělej menší zásahy, průběžně ukládej a nenech retušovat obří plochy najednou.
Stáhnout Oficiální zdroj
1) Otevření fotografie a rychlá příprava
  1. Otevři PhotoPea v prohlížeči.
  2. Načti fotografii jedním ze dvou způsobů:
    • Přetažení – přetáhni JPG z plochy přímo do okna PhotoPea.
    • File → Open – vyber soubor přes dialog.
  3. Poznámka: po načtení může práce pokračovat i při krátkém výpadku internetu (ale po zavření/obnovení stránky je potřeba připojení znovu).
  4. Než začneš velké úpravy, ukládej často (po ořezu a po retuši minimálně vždy jednu verzi navíc).
2) Ořez a narovnání (Crop Tool)
  1. Vyber Crop Tool (zkratka C).
  2. Klikni a tažením vytvoř ořezový rámeček přes celou fotku.
  3. Rotace ořezu:
    • Najeď kurzorem k rohu ořezu a lehce mimo rámeček.
    • Jakmile se kurzor změní na zakřivenou oboustrannou šipku, můžeš otáčet.
  4. Doporučení: ořízni o kousek víc než je nutné.
    • Je těžké trefit rotaci na milimetr.
    • Když ořízneš o trochu víc, nezůstanou ti na okrajích bílé pruhy.
  5. Potvrzení ořezu:
    • Dvojklik do fotky, nebo klik na fajfku (confirm) v horní části.
  6. Volba Delete cropped pixels:
    • Když není zapnutá, oříznuté části se dají později „vytáhnout“ zpět (po dalším ořezu).
    • Když ji zapneš, oříznuté pixely se opravdu odstraní a soubor bývá menší.
3) Uložení pracovní verze po ořezu
  1. Po ořezu si ulož pracovní verzi, ať máš jistotu, že se k ní vrátíš.
  2. Jdi do File (pozor: klikni do „pracovní“ plochy PhotoPea, ne do menu prohlížeče) a zvol uložení do počítače.
  3. Doporučení:
    • Pracovní soubor ukládej jako PSD (zůstane editovatelný).
    • Průběžně ukládej často, zvlášť pokud je aplikace pomalejší.
4) Příprava na retuš: duplikace vrstvy a základ navigace
  1. V panelu Layers zkontroluj, že fotka je na vrstvě Background.
  2. Vytvoř zálohu:
    • Pravý klik na Background → Duplicate Layer.
    • Záložní vrstvu můžeš kdykoliv skrýt/zobrazit ikonou oka.
    • Když vypneš obě vrstvy, uvidíš šachovnici (průhledno) – znamená to, že pod tím nic není.
  3. Pokud ti překáží reklamy, můžeš si pravý panel mírně „odsunout“, aby bylo víc místa na práci.
  4. Navigace:
    • Zoom tool (lupa) – přepínání plus/minus v horní liště voleb nástroje, a tlačítko Fit to area.
    • Mezerník – dočasně „hand“ a posun po fotce.
    • Rychlý zoom bez přepínání nástroje: drž mezerník + Ctrl (PC) / Cmd (Mac) pro přiblížení; přidej ještě Alt/Option pro oddálení.
5) Spot Healing Brush: rychlé odstranění skvrn a prasklin
  1. Vyber nástroj „náplast“ (Spot Healing Brush) – vypadá jako malá náplast.
  2. Klikni a přejeď přes skvrnu (PhotoPea automaticky „nasampluje“ okolí a vadu nahradí).
  3. Velikost štětce:
    • Zkratky: ] zvětší, [ zmenší.
    • V Options bar můžeš nastavit velikost číselně/posuvníkem.
  4. Pravidlo velikosti:
    • Nedělej štětec zbytečně velký – zmenšíš oblast, ze které může nástroj vybírat a výsledek bývá horší.
    • Na praskliny použij malý štětec a dělej to po částech.
  5. Hardness (tvrdost):
    • Vyšší tvrdost = ostřejší okraj, může být vidět „hrana“.
    • Nižší tvrdost = měkký okraj, často lépe splyne s okolím.
  6. Pokud se u okrajů začne objevovat „svit/halo“ (nástroj bere vzorek z kraje papíru), je čas přepnout na Clone Stamp.
6) Clone Stamp: plná kontrola (vzorek + malování)
  1. Vyber Clone Stamp.
  2. Nejdřív nastav zdroj (vzorek):
    • Drž Alt (PC) / Option (Mac) a klikni do čisté oblasti, ze které chceš kopírovat.
    • Na některých zařízeních (např. Chromebook) se modifikační klávesa může chovat jinak – když Alt/Option nefunguje, vyzkoušej alternativně K.
  3. Pak maluj přes vadu – kopíruješ přesně to, co je ve vzorku (včetně struktury a tónu).
  4. Volba Aligned:
    • Zapnuto – zdroj se posouvá spolu s malováním (držení „vzdálenosti“ od štětce).
    • Vypnuto – zdroj zůstává pořád na stejném místě a při každém tahu kopíruješ stále odtud.
  5. Měkký okraj často pomůže:
    • Sniž tvrdost (Hardness), nebo zvol o něco větší štětec.
  6. Kontroluj opakování vzoru:
    • Když zkopíruješ stejné „linky“ nebo textury mockrát, začne to být nápadné.
7) Healing Brush: vzorek jako u klonování, ale s automatickým „prolnutím“
  1. V nástrojích pod „náplastí“ přepni na Healing Brush (není to Spot, ale ten „druhý“).
  2. Stejně jako u Clone Stamp nastav vzorek:
    • Alt/Option + klik do oblasti, kterou chceš použít jako zdroj.
  3. Maluj přes vadu. Rozdíl oproti Clone Stamp:
    • Nezkopíruje to „natvrdo“ – snaží se to sladit jas a barvu cílového místa, takže okraje často lépe splynou.
  4. Kdy co použít:
    • Healing Brush – když chceš, aby to přirozeně splynulo.
    • Clone Stamp – když potřebuješ přesnou kontrolu a Healing dělá duchy/rozmazání (hlavně u hran a linií).
8) Patch Tool: „vyber a přetáhni“ pro větší vady
  1. Vyber Patch Tool (je ve stejné skupině jako retušovací nástroje).
  2. Ohranič vadu výběrem (jako laso) – nakresli uzavřený tvar kolem poškozeného místa.
  3. Klikni dovnitř výběru a přetáhni výběr na čistou oblast, kterou chceš použít jako náhradu.
  4. Pro linie a okraje:
    • Dělej menší výběry – dlouhé křivky se špatně „trefují“.
    • Pokud se objeví svit/halo u okraje, dokonči to Clone Stampem.
  5. Zrušení výběru: Ctrl/Cmd + D (Select → Deselect).
9) Finální uložení a export JPG
  1. Ulož pracovní verzi (doporučení: PSD, aby bylo vidět vrstvy a šlo se vrátit).
  2. Pro odevzdání vyexportuj jako JPG:
    • File → Export as → JPG
    • Nastav Quality na 100 %.
    • Zkontroluj rozměry a cílové umístění, kam se soubor ukládá.
  3. Název souboru:
    • Ulož to s vlastním jménem + doplněním (např. prijmeni_jmeno_retouch.jpg).

Shrnutí

  • Fotku načteš přetažením nebo přes File → Open a po ořezu ukládej průběžně.
  • Crop Tool (C): narovnej, ořízni raději o kousek víc, potvrď dvojklikem; „Delete cropped pixels“ rozhoduje, zda se ořez dá vrátit.
  • Retuš: Spot Healing je nejrychlejší, Healing Brush lépe splývá, Clone Stamp dává plnou kontrolu, Patch je skvělý na větší vady.
  • Pracovní verze ukládej jako PSD, finální odevzdání exportuj jako JPG s kvalitou 100 % a jasným názvem.
