7.1.1 Rastrová grafika — GIMP

Co se naučíš?

Postup přesně podle pracovního listu: instalace GIMPu, nastavení rozhraní, práce s vrstvami, pravítky a textem.

  • Stažení & instalace GIMPu správným nastavením instalátoru.
  • Preferences: jazyk, téma, ikony, systémové zdroje, výchozí obraz.
  • Nový soubor — šablona FullHD, rozlišení X/Y, barevný prostor.
  • Import — Open, přidání složky do rychlého přístupu, Open as Layers.
  • Layout & nástroje — zoom, pravítka (rulery), vrstvy, text a zarovnání.
Tip: Pokud je práce pomalá, uprav v Preferences → System Resources hodnoty Tile Cache Size a Number of threads.
⬇️ Stáhnout balíček obrázků k lekci 1 ⤓ Stáhnout GIMP (oficiální)

  1. Na stránce GIMP Downloads klikni na Download, zvol Windows a pokračuj na Download GIMP directly.
  2. Potvrď bezpečnostní upozornění, v instalátoru zvol Customize, potvrď licenci a cestu k instalaci.
  3. Po dokončení instalace otevři aplikaci.
Obrázky z pracovního listu

Upravit → Preferences → Interface → Language = English. Zavři a znovu otevři aplikaci.

Jazyk rozhraní

Zvol barvu pozadí podle preferencí.

Theme

Zvol zobrazení ikon.

Ikony

  • Minimal number of undo levels, Maximum undo memory
  • Tile Cache Size, Swap compression, Number of threads
System Resources

Nastav šablonu (např. FullHD), X/Y resolution, Color Space a Gamma Precision.

Default Image

V Toolbox odškrtni Use tool groups.

Toolbox

V File → New zvol šablonu FullHD a potvrď.

Nový soubor

  1. Ulož si obrázky do projektové složky obrázky.
  2. File → Open → vyber např. man-4875665.
  3. V Open přidej složku obrázky do rychlého přístupu (ikona +).
  4. File → Open as Layers → vyber víc souborů → Open.
Import

  • Zoom: spodní lišta — přiblížení/oddálení.
  • Ruler & vodítka: přetáhni vodítka z horního/levého pravítka do třetin; kliknutím posun.
Layout & Ruler

  • V panelu Layers vytvoř novou vrstvu:
    • Color tag = šedá, Opacity ≈ 30 %, Width = 600, Height = 200
    • Zarovnej vrstvu podle vodítek
  • Přidej text, zarovnej na střed (případně Move Tool).
Vrstvy & Text

Shrnutí

  • Umíš nainstalovat a spustit GIMP.
  • Nastavíš prostředí (jazyk, téma, ikony, systémové zdroje, výchozí obraz, toolbox).
  • Vytvoříš nový soubor, naimportuješ více snímků jako vrstvy.
  • Pracuješ s pravítky a vodítky, vytvoříš vrstvu s textem a zarovnáš ji.
