🎒 Co je proměnná

Proměnná je pojmenované místo v paměti, kam ukládáme hodnotu, kterou budeme později používat.

jmeno = "Petr"
vek = 18

Proměnná má název a hodnotu. Python si typ odvodí automaticky.

  • Datové typy

    • V Pythonu běžně používáme tyto typy:

      • int – celá čísla (např. 5)
      • float – desetinná čísla (např. 3.14)
      • str – texty (např. "Python")

❓ Proč nemůžu spojit text a číslo pomocí +

V Pythonu (a v mnoha jiných jazycích) má každá proměnná svůj datový typ. Například:

  • str (řetězec, text) – např. "Petr"
  • int (celé číslo) – např. 18

Text a číslo jsou pro počítač úplně jiné věci – stejně jako dílek puzzle a LEGO kostka. Pokud se pokusíš spojit text a číslo pomocí +, Python neví, jestli má číslo převést na text, nebo to spočítat jako čísla, a proto vypíše chybu: 


TypeError: can only concatenate str (not "int") to str

Chceš-li spojit text a číslo dohromady, musíš obě hodnoty nejprve převést na stejný typ!

  • Převod čísla na text: Pomocí str(vek) změníš číslo na text.
  • Použití čárky v print(): Python vše převede na text automaticky a oddělí je mezerou.
  • F-string: Nejmodernější způsob, kdy Python proměnnou sám převede na text ve složených závorkách f"Věk: {vek}".

Proč to tak je?

  • Každý datový typ se v paměti ukládá jinak a Python to má rád přehledné.
  • Python tě raději upozorní na chybu, než aby automaticky měnil typy (na rozdíl od některých jiných jazyků, např. JavaScript).
  • Díky tomu je kód přehlednější a méně náchylný na nechtěné chyby.

Příklad chyby: print("Věk: " + vek) ⟶ chyba!
Správně:
  • print("Věk: " + str(vek))
  • print("Věk:", vek)
  • print(f"Věk: {vek}")

Proto se vždy ujisti, že spojované hodnoty mají stejný typ – nejčastěji text (str).

🔀 Podmínky v Pythonu: if, elif, else

Podmínka v Pythonu umožňuje rozhodnout, zda se určitý blok kódu provede, podle toho, jestli je nějaké tvrzení pravdivé nebo ne.
Nejčastěji používáme klíčová slova if, elif a else: 


x = 10
if x > 5:
    print("x je větší než 5")
elif x == 5:
    print("x je rovno 5")
else:
    print("x je menší než 5")
  • if znamená „pokud“. Když je podmínka za if pravdivá, vykoná se blok kódu pod ním.
  • elif znamená „jinak, pokud“. Je to zkratka pro else if. Používáme ho, když chceme více různých možností. Python je projde shora dolů a provede první blok, jehož podmínka platí.
  • else znamená „jinak“. Spustí se jen tehdy, když neplatila žádná předchozí podmínka.
📏 Odsazení (mezery na začátku řádku)

Odsazení je v Pythonu povinné! Říká, které příkazy patří „pod“ podmínku. Je to obdoba závorek v jiných jazycích.
Doporučení: používej 4 mezery pro každou úroveň.
Chybný zápis (chybí odsazení): 


if x > 5:
print("x je větší než 5")    # Chyba! Není odsazeno.

Správně: 


if x > 5:
    print("x je větší než 5")  # Správně – řádek podmínky je odsazený
Tip: Odsazení určuje, co patří do bloku (za if/elif/else). Všechny řádky se stejným odsazením patří do jednoho bloku. Pokud chceš víc příkazů v podmínce, všechny musejí být odsazené stejně.
📍 Proč se píše : (dvojtečka)?

Dvojtečka na konci řádku za if, elif nebo else říká Pythonu: „teď začíná blok příkazů“.
Musí tam být vždy – je to součást syntaxe.
Příklad: if x > 5:
Pokud dvojtečku zapomeneš, Python vypíše chybu syntaxe.

📌 Proč u podmínky nejsou závorky?

Python je jednoduchý – nepotřebuješ kulaté závorky kolem podmínky. Prostě napíšeš podmínku rovnou za if, např. if x > 5:.

🤔 Co znamená == a proč nestačí jedno rovnítko?

  • = (jedno rovnítko) znamená přiřazení – použiješ ho, když do proměnné ukládáš hodnotu. (např. x = 5)
  • == (dvojité rovnítko) znamená porovnání – ptáš se, jestli má x určitou hodnotu. (např. x == 5)
Příklad: 

x = 10       # přiřazení
if x == 10:  # porovnání
    print("x je 10")
Pozor: Pokud místo == omylem použiješ = v podmínce, Python vypíše chybu (SyntaxError).

🧠 Jak to celé funguje v programu?
  1. Python vyhodnotí podmínku za if – pokud je pravda, provede blok podmínky a dál už pokračuje za celou podmínku.
  2. Pokud if nebyla pravda, zkusí elif (můžeš mít i víc elif za sebou).
  3. Pokud nebyla pravda ani žádná elif, provede se else (pokud je uveden).
  4. Provede se vždy jen jeden blok – první, jehož podmínka je splněná.
Rychlý přehled:
  • Odsazení určuje, co patří do bloku (4 mezery = 1 úroveň).
  • Dvojtečka : vždy za if, elif, else!
  • if – když platí podmínka, elif – další možnost, else – všechno ostatní.
  • == = porovnání, = = přiřazení.
