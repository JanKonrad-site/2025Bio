4.3.1. Textová adventura s výběrem směru

Co se naučíš?

V této kapitole si vytvoříš jednoduchou textovou adventuru, ve které bude hráč volit směr a dostane odpověď podle své volby. Zároveň se naučíš základy podmínek v jazyce Python a vstupu od uživatele pomocí input().

  • Funkce input(): Získání vstupu od uživatele.
  • Podmínky (if, elif, else): Reakce programu na volbu hráče.
Tip: Tento typ programu je skvělý základ pro větší příběhové hry, kde si můžeš vymyslet vlastní svět.

Jak funguje rozhodovací strom

Pomocí input() získáš odpověď od uživatele a pomocí podmínek if, elif a else rozhodneš, co se má v programu stát.

  • Rozhodovací větvení
    • Uživatel zadá písmeno a ty na základě jeho odpovědi rozhodneš, který blok kódu se vykoná: 
      volba = input("Zadej a, b, c, nebo d: ")
if volba == "a":
    print("Šel jsi na sever.")
elif volba == "b":
    print("Šel jsi na jih.")
else:
    print("Neudělal jsi nic.")

💡 Hint: Nezapomeň zkontrolovat odsazení! Každý blok if musí mít správné zarovnání (tabulátor nebo mezery).

Úkoly:

  • 1. Vytvoř vlastní příběh, který začíná probuzením na ostrově.
  • 2. Doplň 4 volby (a, b, c, d) a ke každé vymysli odpověď.
  • 3. Pokud uživatel zadá neplatnou volbu, napiš mu zprávu.

Jak funguje podmínka if

Pomocí if kontrolujeme, co uživatel zadal. Pokud to odpovídá předem určené hodnotě (např. "a"), provede se příslušný kód. Pokud ne, použijeme elif nebo else.

if volba == "a":
    print("Našel jsi poklad!")
elif volba == "b":
    print("Utékl jsi před drakem!")
else:
    print("Zůstal jsi na místě a usnul.")

Tip: if = když, elif = jinak když, else = jinak
