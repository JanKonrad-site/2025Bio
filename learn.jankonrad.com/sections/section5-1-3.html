3.2.3 SQLite v Pythonu – miniaplikace „Moje filmy“ (skládání z bloků)

V této lekci máš jeden soubor movies_app.py. Do něj postupně přidáváš malé bloky kódu (funkce). Menu a hlavní smyčku si skládáš ručně – přesně tak, aby sis uvědomoval(a), co kam patří.

Co se v této lekci naučíš?

  • Databázové připojení – co vrací sqlite3.connect() a proč tomu říkáme conn.
  • Vytvoření tabulky – CREATE TABLE IF NOT EXISTS a proč to chceme.
  • Vložení dat – INSERT a parametry ?.
  • Čtení dat – SELECT, fetchall() a výpis přes for.
  • commit() a close() – kdy se ukládá do souboru a proč spojení zavíráme.
Pravidlo lekce: Po každém bloku program spusť a ověř, že přesně chápeš, co se změnilo.

Workflow (jak pracuješ)

  1. Vytvoř jeden soubor: movies_app.py.
  2. Vlož Skeleton (Krok 0).
  3. Pak už jen kopíruj jednotlivé BLOKY (Krok 1, 2, 3…) do části „FUNKCE“.
  4. Do části „MAIN“ přidávej ručně řádky podle instrukcí.
Kam co patří: Funkce dáváš nad MAIN. Hlavní smyčka (menu) je až v MAIN.
Krok 0: Skeleton souboru (jediné, co kopíruješ celé)

Teď opravdu kopíruješ celý soubor. Dál už jen bloky.

Krok 1: Připojení + vytvoření tabulky (connect + init_db)

Vlož následující blok do části # [BLOCKS_GO_HERE]. To je „motor“: umíme se připojit a připravit tabulku.

BLOCK 1 – connect() + init_db() 
def connect():
    # str(DB_FILE) = jistota, že sqlite3 dostane obyčejný text (cestu)
    return sqlite3.connect(str(DB_FILE))

def init_db():
    conn = connect()

    conn.execute("""
    CREATE TABLE IF NOT EXISTS movies (
        id    INTEGER PRIMARY KEY AUTOINCREMENT,
        title TEXT    NOT NULL,
        year  INTEGER
    );
    """)

    conn.commit()
    conn.close()

Teď do MAIN (místo pass) ručně vlož tento test:

Krok 2: Vložení filmu (INSERT) – funkce add_movie()

Vlož funkci do části „FUNKCE“. Potom v MAIN uděláš ruční test.

BLOCK 2 – add_movie() 
def add_movie():
    title = input("Název filmu: ").strip()
    year_text = input("Rok (ENTER = nevyplnit): ").strip()
    year = int(year_text) if year_text else None

    conn = connect()
    conn.execute("INSERT INTO movies (title, year) VALUES (?, ?);", (title, year))
    conn.commit()
    conn.close()

    print("Uloženo.")

BLOCK 3 – fetch_movies() + print_movies() 
def fetch_movies():
    conn = connect()
    rows = conn.execute("SELECT id, title, year FROM movies ORDER BY id DESC;").fetchall()
    conn.close()
    return rows

def print_movies(rows):
    print("Obsah databáze:")
    for (movie_id, title, year) in rows:
        yr = year if year is not None else "?"
        print(f"#{movie_id} | {title} ({yr})")

Do MAIN si ručně vlož test:

Krok 4: Info o DB (sqlite_master + sqlite_sequence)
BLOCK 4 – list_tables() + show_db_info() 
def list_tables():
    conn = connect()
    rows = conn.execute("SELECT name FROM sqlite_master WHERE type='table';").fetchall()
    conn.close()
    return [name for (name,) in rows]

def show_db_info():
    print("DB path:", DB_FILE)
    print("DB exists:", DB_FILE.exists())
    print("Tables:", list_tables())

Nápověda: co má dělat která volba

  • "1" → add_movie()
  • "2" → rows = fetch_movies() + print_movies(rows)
  • "3" → show_db_info()
  • "0" → break
  • jinak → print("Neplatná volba.")

Hotový stav – kontrolní seznam

  • Program se spustí a ukáže menu.
  • Volba 1 vloží film a napíše „Uloženo.“
  • Volba 2 vypíše filmy ve formátu #id | title (year).
  • Volba 3 vypíše cestu k DB + tabulky.
  • Volba 0 program ukončí.
