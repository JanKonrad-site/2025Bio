3.2.3 SQLite v Pythonu – miniaplikace „Moje filmy“ (skládání z bloků)

V této lekci máš jeden soubor movies_app.py. Do něj postupně přidáváš malé bloky kódu (funkce). Menu a hlavní smyčku si skládáš ručně – přesně tak, aby sis uvědomoval(a), co kam patří.

Workflow (jak pracuješ)

  1. Vytvoř jeden soubor: movies_app.py.
  2. Vlož Skeleton (Krok 0).
  3. Pak už jen kopíruj jednotlivé BLOKY (Krok 1, 2, 3…) a vkládej je na správné místo.
  4. Po každém bloku program spusť a ověř výstup.
Pravidlo: Funkce vkládej do části „FUNKCE (tady přidáváš bloky)“. Hlavní běh patří do části „MAIN“.
Krok 0: Skeleton souboru (základ, do kterého budeš vkládat bloky)

Teď opravdu kopíruješ celý soubor. To je jediná výjimka. Dál už kopíruješ jen bloky.

Krok 1: Přidej blok „Databázové připojení + vytvoření tabulky“

Vlož následující blok do části # [BLOCKS_GO_HERE] (nad MAIN). Tohle je „motor“: umíme se připojit a připravit tabulku.

BLOCK 1 – connect() + init_db() 
def connect():
    # str(DB_FILE) = jistota, že sqlite3 dostane obyčejný text (cestu)
    return sqlite3.connect(str(DB_FILE))

def init_db():
    # 1) připojit
    conn = connect()

    # 2) poslat SQL příkaz (vytvoří tabulku, pokud ještě neexistuje)
    conn.execute("""
    CREATE TABLE IF NOT EXISTS movies (
        id    INTEGER PRIMARY KEY AUTOINCREMENT,
        title TEXT    NOT NULL,
        year  INTEGER
    );
    """)

    # 3) potvrdit změny
    conn.commit()

    # 4) zavřít spojení
    conn.close()

Teď si do MAIN části (místo pass) napiš ručně 2 řádky: zavolej init_db() a vypiš DB cestu.

MAIN – test pro Krok 1

Krok 2: Přidej blok „Vložení filmu“ (INSERT)

Vlož blok do části funkcí. V MAIN pak ručně otestuješ, že vložení funguje.

BLOCK 2 – add_movie() 
def add_movie():
    title = input("Název filmu: ").strip()
    year_text = input("Rok (ENTER = nevyplnit): ").strip()
    year = int(year_text) if year_text else None

    conn = connect()
    conn.execute("INSERT INTO movies (title, year) VALUES (?, ?);", (title, year))
    conn.commit()
    conn.close()

    print("Uloženo.")

def fetch_movies():
    conn = connect()
    rows = conn.execute("SELECT id, title, year FROM movies ORDER BY id DESC;").fetchall()
    conn.close()
    return rows

def print_movies(rows):
    print("Obsah databáze:")
    for (movie_id, title, year) in rows:
        yr = year if year is not None else "?"
        print(f"#{movie_id} | {title} ({yr})")

Do MAIN si ručně dej test: načti řádky a vytiskni je.

def list_tables():
    conn = connect()
    rows = conn.execute("SELECT name FROM sqlite_master WHERE type='table';").fetchall()
    conn.close()
    return [name for (name,) in rows]

def show_db_info():
    print("DB path:", DB_FILE)
    print("DB exists:", DB_FILE.exists())
    print("Tables:", list_tables())

Krok 6: Hlavní smyčka aplikace (while True) – sem už lepíš ručně jen řádky

Teď přijde hlavní skládání: do MAIN si ručně poskládáš smyčku. Nechci ti sem dát hotový celý MAIN, protože pointa je, že ty sám/sama propojíš volby s funkcemi.

Postup skládání MAIN:
  1. Na začátek MAIN dej init_db().
  2. Pak napiš while True:.
  3. Uvnitř smyčky zavolej menu().
  4. Načti volbu: choice = input(...).strip().
  5. Přidej větvení pro volby 1/2/3/0.

Šablona smyčky (kopíruj, zbytek doplníš ručně):

MAIN – šablona smyčky 
init_db()

while True:
    menu()
    choice = input("Volba: ").strip()

    # sem doplň if/elif/else pro volby

Nápověda: co má dělat která volba

  • "1" → add_movie()
  • "2" → rows = fetch_movies() + print_movies(rows)
  • "3" → show_db_info()
  • "0" → break
  • jinak → print("Neplatná volba.")

Hotový stav – kontrolní seznam

  • Program se spustí a ukáže menu.
  • Volba 1 vloží film a napíše „Uloženo.“
  • Volba 2 vypíše filmy ve formátu #id | title (year).
  • Volba 3 vypíše cestu k DB + tabulky.
  • Volba 0 program ukončí.
