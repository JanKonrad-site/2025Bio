3.6.0 Rezervační systém v Pythonu – konzole + SQLite (skládání z bloků)

Vytvoříš jednoduchý rezervační systém, který běží v konzoli. Data se ukládají do databáze SQLite (reservations.db) vedle skriptu. Aplikace umí rezervaci přidat a vypsat.

Co se v této lekci naučíš?

  • Databázový soubor – kde je DB uložená a jak to ověřit (Path, __file__, exists()).
  • Tabulka a záznam – jak vytvořit tabulku pro rezervace (CREATE TABLE).
  • Zápis dat – jak vložit rezervaci do tabulky (INSERT + ? parametry).
  • Čtení dat – jak načíst rezervace a vypsat je (SELECT, fetchall(), for).
  • Menu aplikace – jak to celé spojit do while True smyčky.
Formáty vstupu: datum piš jako YYYY-MM-DD (např. 2026-04-10), čas jako HH:MM (např. 14:30). Zatím nepoužíváme datetime knihovny – držíme to jednoduché.

Krok 1: Databázový motor (connect + init_db)

Teď přidáš „motor“: umíme se připojit k databázi a vytvořit tabulku reservations. Vlož blok do části # [BLOCKS_GO_HERE].

BLOCK 1 – connect() + init_db() 
def connect():
    # str(DB_FILE) = jistota, že sqlite3 dostane obyčejný text cesty
    return sqlite3.connect(str(DB_FILE))

def init_db():
    conn = connect()

    conn.execute("""
    CREATE TABLE IF NOT EXISTS reservations (
        id      INTEGER PRIMARY KEY AUTOINCREMENT,
        name    TEXT    NOT NULL,
        date    TEXT    NOT NULL,
        arrival TEXT    NOT NULL,
        departure TEXT  NOT NULL
    );
    """)

    conn.commit()
    conn.close()

def add_reservation():
    name = input("Jméno: ").strip()
    date = input("Datum (YYYY-MM-DD): ").strip()
    arrival = input("Příchod (HH:MM): ").strip()
    departure = input("Odchod (HH:MM): ").strip()

    conn = connect()
    conn.execute(
        "INSERT INTO reservations (name, date, arrival, departure) VALUES (?, ?, ?, ?);",
        (name, date, arrival, departure)
    )
    conn.commit()
    conn.close()

    print("Uloženo.")

BLOCK 3 – fetch_reservations() + print_reservations() 
def fetch_reservations():
    conn = connect()
    rows = conn.execute("""
        SELECT id, name, date, arrival, departure
        FROM reservations
        ORDER BY date ASC, arrival ASC;
    """).fetchall()
    conn.close()
    return rows

def print_reservations(rows):
    if not rows:
        print("Žádné rezervace.")
        return

    print("Rezervace:")
    for (rid, name, date, arrival, departure) in rows:
        print(f"#{rid} | {name} | {date} | {arrival}–{departure}")

def show_reservations_for_date():
    d = input("Datum (YYYY-MM-DD): ").strip()

    conn = connect()
    rows = conn.execute("""
        SELECT id, name, date, arrival, departure
        FROM reservations
        WHERE date = ?
        ORDER BY arrival ASC;
    """, (d,)).fetchall()
    conn.close()

    print_reservations(rows)

Path / resolve / with_name / exists

  • __file__ = cesta k aktuálnímu .py souboru
  • Path(__file__) = objekt cesty
  • resolve() = absolutní „vyřešená“ cesta
  • with_name("reservations.db") = stejná složka, jiný název souboru
  • exists() = ověření, že soubor opravdu existuje

CREATE TABLE IF NOT EXISTS

CREATE TABLE vytvoří tabulku. IF NOT EXISTS je pojistka, aby to šlo spustit opakovaně.

commit() a close()

  • commit() = potvrdí změny (CREATE/INSERT/DELETE) do databáze
  • close() = ukončí spojení k DB (dobrý zvyk)

Proč str(DB_FILE)

DB_FILE je objekt Path. str(DB_FILE) ho převede na textovou cestu.

INSERT a parametry ?

Parametry posíláš zvlášť, aby se SQL neskládalo ručně a bylo bezpečné:

conn.execute(
  "INSERT INTO reservations (name, date, arrival, departure) VALUES (?, ?, ?, ?);",
  (name, date, arrival, departure)
)

for cyklus + rozbalení tuple

Každý řádek z DB je tuple (n-tice). Rozbalíš ho přímo v hlavičce cyklu:

for (rid, name, date, arrival, departure) in rows:
    print(rid, name)

SQL: SELECT / FROM / ORDER BY

  • SELECT = které sloupce chci
  • FROM = z jaké tabulky
  • ORDER BY = jak seřadit výsledek

Proč (d,) a ne (d)

(d) je jen d v závorkách (není to tuple). Jednoprvkový tuple je (d,) – čárka je rozhodující.

while True + break

while True běží pořád. Ukončíš ho break (např. při volbě 0).

if / elif / else v menu

if choice == "1":
    add_reservation()
elif choice == "2":
    rows = fetch_reservations()
    print_reservations(rows)
elif choice == "0":
    break
else:
    print("Neplatná volba.")
