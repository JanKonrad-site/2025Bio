4.1.8 Závěrečný test – podmínky, porovnání a logické operátory

Jak test funguje?

  • Klikni na jednu odpověď u každé otázky.
  • Správná odpověď se vyhodnotí pomocí funkce checkAnswer(this, ...) (stejně jako v CSS testu).
  • Otázky vycházejí z kapitol 4.1.2a–d: podmínky, elif, modulo, logické operátory a pořadí vyhodnocování.
Doporučení: před testem si projdi „Treasure Island“ a příklady s horskou dráhou / pizzou.

Otázka 1: Co vyhodnotí následující výraz v Pythonu?

not 5 == 5

Nápověda: == má přednost před not.

Otázka 2: Co vyhodnotí následující výraz?

False or True or False

Otázka 3: Co vytiskne následující kód?

a = 5
b = 7
 
if a >= b and a != b:
    print("A")
elif not a >= b and a != b:
    print("B")
else:
    print("C")

Otázka 4: Co vytiskne tento kód?

number_to_check = 12

if number_to_check % 2 == 0:
    print("even")
else:
    print("odd")

Nápověda: modulo % vrací zbytek po dělení.

Otázka 5: Jaký bude výstup následujícího programu?

age = 20
bill = 0

if age < 12:
    bill = 5
    print("Dětský lístek.")
elif age <= 18:
    bill = 7
    print("Studentský lístek.")
else:
    bill = 12
    print("Dospělý lístek.")

if age >= 60:
    bill = 6
    print("Sleva pro seniory.")

print(bill)

Nápověda: if/elif/else z bloku vybere jen jednu větev, ale další samostatné if se kontroluje zvlášť.

Otázka 6: Které tvrzení o if / elif / else je správné?

Co bys po tomto testu měl(a) umět?

  • Rozhodnout, jak se vyhodnotí jednoduché i složené logické výrazy (and, or, not).
  • Přečíst a pochopit kód s if / elif / else a vnořenými podmínkami.
  • Použít % (modulo) k testu sudosti/lichosti.
  • Vědět, jak funguje více nezávislých if za sebou.
