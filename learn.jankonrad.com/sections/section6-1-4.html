6.1.2 GUI v Pythonu – základní pojmy před prvním oknem

Než vznikne první okno aplikace, je potřeba rozumět několika základním pojmům. Bez nich vypadají zápisy jako import tkinter as tk, tk.Tk() nebo root.title(...) jen jako něco, co se opisuje zpaměti.

V této části se proto ještě nebude stavět formulář ani tabulka. Nejdřív se opravdu poctivě vysvětlí, co znamená modul, import, alias, třída, objekt, metoda a co znamená tečka v zápisech jako tk.Tk() nebo root.title().

Co se v této části přesně objasní

  • Co je modul.
  • Co dělá příkaz import.
  • Co znamená as tk.
  • Co je ttk.
  • Co je třída.
  • Co je objekt.
  • Co je metoda.
  • Co znamená tečka v zápisu tk.Tk nebo root.title.
  • Jak číst první čtyři řádky GUI programu opravdu slovo od slova.
Hlavní cíl:
Neopíšeme první GUI okno jen mechanicky, ale pochopíme, co přesně Python na každém řádku dělá.
Krok 1: co je modul

Modul je soubor s Python kódem, který obsahuje něco užitečného, co lze použít i v jiném programu.

V modulu mohou být například:

  • funkce,
  • třídy,
  • proměnné,
  • další pomocné nástroje.

Když se řekne, že tkinter je modul, znamená to: Python už má připravený kus kódu, který umí pracovat s grafickým oknem, tlačítky, textovými poli a dalšími GUI prvky.

Zjednodušeně:
Modul je hotová výbava, kterou si program může načíst a používat.
Krok 2: co dělá import

Příkaz import znamená: načti modul do programu, abych ho mohl používat.

Například:

Základní import 
import tkinter

Tím se do programu načte modul tkinter.

Od této chvíle může program používat to, co je uvnitř modulu tkinter.

Přesně:
import nic nekreslí a nic neotevírá. Jen zpřístupní modul pro další použití.
Krok 3: co znamená import tkinter as tk

Tohle je jeden z nejčastějších zápisů v tkinteru:

Import s aliasem 
import tkinter as tk

Rozbor slovo od slova

  • import = načti modul.
  • tkinter = název modulu.
  • as = pod jiným jménem.
  • tk = nové kratší jméno modulu.

Celý řádek tedy znamená:

Načti modul tkinter a v tomto programu mu říkej zkráceně tk.

Proč se to dělá? Protože tkinter je delší název a zkrácené tk se pohodlněji píše.

Bez aliasu

import tkinter

root = tkinter.Tk()

S aliasem

import tkinter as tk

root = tk.Tk()

Obojí znamená totéž. Druhá varianta je jen kratší a přehlednější.

Krok 4: co je ttk

Často se používá také tento zápis:

Import ttk 
from tkinter import ttk

Rozbor slovo od slova

  • from = z něčeho vezmi,
  • tkinter = z modulu tkinter,
  • import = načti,
  • ttk = část modulu tkinter jménem ttk.

ttk je sada modernějších grafických prvků, například hezčí tabulka nebo stylovanější widgety.

V praxi to znamená: modul tkinter obsahuje více částí a jednou z nich je právě ttk.

Příklad použití

tree = ttk.Treeview(...)

Tím se používá prvek Treeview, který je uvnitř ttk.

Krok 5: co je třída

Třída je v programování něco jako plán nebo šablona, podle které se vytvářejí konkrétní objekty.

Představa z běžného života:

  • třída = plán domu,
  • objekt = konkrétní postavený dům.

V tkinteru je například Tk třída. Ta představuje „typ hlavního okna aplikace“.

Třída tedy není konkrétní okno.
Je to předpis, jak se takové okno vytvoří.
Krok 6: co je objekt

Objekt je konkrétní vytvořená věc podle třídy.

Když třída představuje plán, objekt je konkrétní výsledek podle toho plánu.

Například: 

root = tk.Tk()

znamená:

  • Tk = třída,
  • tk.Tk() = vytvoření objektu podle této třídy,
  • root = proměnná, ve které je tento objekt uložený.
Přesně:
root je konkrétní hlavní okno aplikace.
Krok 7: co znamená tečka

Tečka v Pythonu znamená: vezmi něco, co patří k něčemu jinému.

Tečka tedy propojuje celek a jeho část.

Příklad 1

tk.Tk

To znamená: „vezmi Tk z modulu tk“

  • tk = modul tkinter pod zkráceným jménem,
  • Tk = třída uvnitř tohoto modulu.

Příklad 2

root.title

To znamená: „vezmi title z objektu root“

  • root = konkrétní objekt, tedy okno,
  • title = metoda, která patří tomuto objektu.
Tečka neznamená matematiku.
Znamená přístup k něčemu, co patří modulu nebo objektu.
Krok 8: co je metoda

Metoda je funkce, která patří konkrétnímu objektu nebo třídě.

Například: 

root.title("Rezervační systém")

Tady je title metoda objektu root.

To znamená: hlavní okno root má schopnost nastavit svůj titulek a dělá to právě metodou title(...).

Rozdíl mezi funkcí a metodou

  • funkce stojí samostatně, například print("Ahoj"),
  • metoda patří nějakému objektu, například root.title(...).
Zjednodušeně:
metoda je „činnost“, kterou umí konkrétní objekt.
Krok 9: rozbor zápisu root = tk.Tk() opravdu po částech
Rozebíraný řádek 
root = tk.Tk()

Rozbor slovo od slova

  • root = proměnná, do které se něco uloží.
  • = = přiřazení, tedy „ulož výsledek doprava do jména vlevo“.
  • tk = zkrácené jméno modulu tkinter.
  • . = vezmi něco z modulu tk.
  • Tk = třída uvnitř modulu tkinter.
  • () = zavolej, tedy vytvoř objekt podle této třídy.

Celý řádek tedy znamená:

Vytvoř hlavní okno aplikace podle třídy Tk a ulož ho do proměnné root.
Krok 10: rozbor zápisu root.title("Rezervační systém")
Rozebíraný řádek 
root.title("Rezervační systém")

Rozbor slovo od slova

  • root = objekt hlavního okna.
  • . = vezmi něco, co patří objektu root.
  • title = metoda objektu root.
  • ("Rezervační systém") = zavolej tuto metodu a předej jí text jako argument.

Celý řádek znamená:

Zavolej na hlavním okně metodu, která nastaví jeho titulek na „Rezervační systém“.
Krok 11: první čtyři řádky GUI programu rozebrané dohromady
První minimální GUI 
import tkinter as tk

root = tk.Tk()
root.title("Rezervační systém")
root.mainloop()

Co ty řádky dohromady znamenají

  1. Načti modul tkinter a říkej mu zkráceně tk.
  2. Pomocí třídy Tk vytvoř hlavní okno a ulož ho do root.
  3. Nastav tomuto oknu titulek.
  4. Spusť hlavní smyčku aplikace, aby okno zůstalo otevřené a reagovalo na události.

V této chvíli ještě neexistují tlačítka ani vstupní pole. Existuje jen samotné okno.

Krok 12: co je teď opravdu pevné
  • tkinter je modul pro GUI v Pythonu.
  • import načítá modul do programu.
  • as tk vytváří zkrácené jméno modulu.
  • ttk je část tkinteru s dalšími widgety.
  • Tk je třída pro hlavní okno aplikace.
  • root je konkrétní objekt vytvořený podle této třídy.
  • Tečka znamená přístup k něčemu, co patří modulu nebo objektu.
  • title je metoda objektu root.
