4.1.7 Funkce krok za krokem – od základů až k mini-projektu Rozpočet LITE

Začneme úplně jednoduše a budeme přidávat stavební kameny: definice → parametry → návratová hodnota → funkce jako hodnoty → bezpečný vstup → n-tice a výpis → dispatcher → mini-projekt Rozpočet LITE.

def pozdrav():
    print("Ahoj")

pozdrav()  # ← volání: závorky spustí kód uvnitř
  • def pozdrav(): vytvoří funkci (ale nespustí ji).
  • pozdrav() ji spustí. Bez závorek je to jen odkaz na funkci.
Mini-úkol: Napiš funkci program_bezi(), která dvakrát vytiskne „Program běží“, a zavolej ji.

def soucet(a, b):          # a, b = parametry
    print(a + b)

soucet(2, 3)               # poziční argumenty
soucet(a=2, b=3)           # pojmenované argumenty
soucet(b=10, a=1)          # pořadí už nevadí

def ahoj(jmeno="Student"): # výchozí argument
    print("Ahoj", jmeno)

ahoj()          # Ahoj Student
ahoj("Jan")     # Ahoj Jan
  • Parametr = „škatulka“ ve funkci; argument = hodnota při volání.
  • Pojmenované argumenty zvyšují čitelnost.
  • Výchozí hodnoty dělají parametr volitelný.
Pozor: jako výchozí hodnotu nepoužívej měnitelná data (list, dict) – viz modál.
Mini-úkol: Napiš pozdrav(jmeno, vek) a zavolej ji pozičně i pojmenovaně.

def soucet(a, b):
    return a + b     # data vrátím (můžu je dál použít)

x = soucet(5, 7)
y = soucet(10, 15)
print(x + y)

def min_max(a, b):   # víc hodnot = n-tice (tuple)
    return min(a,b), max(a,b)

mensi, vetsi = min_max(8, 3)   # rozbalování (unpacking)
print(mensi, vetsi)
  • return → data žijí dál; print jen zobrazí.
  • Vícenásobný návrat = jedna n-tice → na levé straně ji rozbalím do proměnných.
Mini-úkol: Vezmi input() (vrací str), funkce ho vrátí, mimo funkci vypiš.

def soucet(a,b): return a + b
def rozdil(a,b): return a - b

F = soucet     # uložím odkaz (bez závorek!)
G = rozdil

print(F(10,5)) # 15
print(G(10,5)) # 5

operace = {"plus": soucet, "minus": rozdil}
print( operace["plus"](3, 9) )   # 12

Klíč pro dispatcher – mapování volby na funkci bez mega-if/elif.

Mini-úkol: Přidej soucin(a,b) pod klíč "krat".

def nacti_int(vyzva, min_value=None, max_value=None):
    """Bezpečně načte celé číslo s volitelnými hranicemi."""
    while True:
        try:
            hodnota = int(input(vyzva))
        except ValueError:
            print("Zadejte prosím celé číslo."); continue
        if min_value is not None and hodnota < min_value:
            print(f"Hodnota musí být ≥ {min_value}."); continue
        if max_value is not None and hodnota > max_value:
            print(f"Hodnota musí být ≤ {max_value}."); continue
        return hodnota

def nacti_kladnou_castku(vyzva):
    return nacti_int(vyzva, min_value=0)
  • Jedna univerzální funkce pro převod i validaci.
Mini-úkol: Načti věk (0–130) přes nacti_int(...,0,130).

def vypis_polozek(polozky, nadpis):
    print(f"\n{nadpis}:")
    if not polozky:
        print("  (žádné položky)"); return 0
    suma = 0
    for i, (nazev, castka) in enumerate(polozky, start=1):
        print(f"  {i:>2}. {nazev} — {castka} Kč")
        suma += castka
    print("  ─────────────────────────")
    print(f"  Součet: {suma} Kč")
    return suma

# ukázka:
polozky_demo = [("Výplata", 30000), ("Kafe", -60), ("Kniha", -500)]
vypis_polozek(polozky_demo, "PŘEHLED")
  • Položka jako n-tice (název, částka).
  • enumerate(..., start=1) pro hezké číslování.
Mini-úkol: Rozděl data na prijmy a vydaje, spočti zůstatek = příjem − výdaje.

def akce_a(): print("Akce A")
def akce_b(): print("Akce B")

DISPATCH = {"1": ("Udělej A", akce_a),
            "2": ("Udělej B", akce_b),
            "3": ("Konec",    None)}

def vytiskni_menu():
    print("\n=== MENU ===")
    for klic, (nazev, _) in DISPATCH.items():
        print(f"{klic}) {nazev}")

while True:
    vytiskni_menu()
    volba = input("Vyber: ").strip()
    nazev, funkce = DISPATCH.get(volba, (None, None))
    if funkce is None:
        if volba == "3": print("Konec."); break
        print("Neplatná volba."); continue
    funkce()
  • Místo if/elif udržujeme mapu „volba → funkce”.
  • .get(...,(None,None)) elegantně kryje neplatnou volbu.
Mini-úkol: Přidej třetí akci, která požádá o jméno a pozdraví.

print("Vítejte v aplikaci Rozpočet LITE")

# Datové struktury: seznamy n-tic (název, částka)
prijmy, vydaje = [], []

# ---------- Bezpečné načítání ----------
def nacti_int(vyzva, min_value=None, max_value=None):
    while True:
        try:
            hodnota = int(input(vyzva))
        except ValueError:
            print("Zadejte prosím celé číslo."); continue
        if min_value is not None and hodnota < min_value:
            print(f"Hodnota musí být ≥ {min_value}."); continue
        if max_value is not None and hodnota > max_value:
            print(f"Hodnota musí být ≤ {max_value}."); continue
        return hodnota

def nacti_kladnou_castku(vyzva): return nacti_int(vyzva, min_value=0)

# ---------- Výpisy ----------
def vypis_polozek(polozky, nadpis):
    print(f"\n{nadpis}:")
    if not polozky:
        print("  (žádné položky)"); return 0
    suma = 0
    for i, (nazev, castka) in enumerate(polozky, start=1):
        print(f"  {i:>2}. {nazev} — {castka} Kč")
        suma += castka
    print("  ─────────────────────────")
    print(f"  Součet: {suma} Kč")
    return suma

def vypis_prehled():
    p = vypis_polozek(prijmy, "PŘÍJMY")
    v = vypis_polozek(vydaje, "VÝDAJE")
    print("\n=== SOUHRN ===")
    print(f"Příjem celkem: {p} Kč")
    print(f"Výdaje celkem: {v} Kč")
    print(f"Zůstatek: {p - v} Kč\n")

# ---------- Akce ----------
def akce_pridat_prijem():
    nazev = input("Název příjmu: ").strip() or "Nepojmenovaný příjem"
    castka = nacti_kladnou_castku("Částka (Kč): ")
    prijmy.append((nazev, castka))
    print("✅ Příjem přidán.")

def akce_pridat_vydej():
    nazev = input("Název výdaje: ").strip() or "Nepojmenovaný výdaj"
    castka = nacti_kladnou_castku("Částka (Kč): ")
    vydaje.append((nazev, castka))
    print("✅ Výdaj přidán.")

def akce_odebrat_prijem():
    if not prijmy:
        print("Žádné příjmy k odebrání."); return
    vypis_polozek(prijmy, "PŘÍJMY")
    idx = nacti_int("Zadejte číslo položky k odebrání: ", 1, len(prijmy))
    odebrano = prijmy.pop(idx - 1)
    print(f"🗑️ Odebrán příjem: {odebrano[0]} — {odebrano[1]} Kč")

def akce_odebrat_vydej():
    if not vydaje:
        print("Žádné výdaje k odebrání."); return
    vypis_polozek(vydaje, "VÝDAJE")
    idx = nacti_int("Zadejte číslo položky k odebrání: ", 1, len(vydaje))
    odebrano = vydaje.pop(idx - 1)
    print(f"🗑️ Odebrán výdaj: {odebrano[0]} — {odebrano[1]} Kč")

def akce_vypis(): vypis_prehled()

def akce_ukoncit():
    print("Ukončuji aplikaci…")
    return "EXIT"

# ---------- Dispatcher ----------
MAIN_DISPATCH = {
    "1": ("Přidat příjem",   akce_pridat_prijem),
    "2": ("Přidat výdaj",    akce_pridat_vydej),
    "3": ("Odebrat příjem",  akce_odebrat_prijem),
    "4": ("Odebrat výdaj",   akce_odebrat_vydej),
    "5": ("Výpis přehledu",  akce_vypis),
    "6": ("Ukončit",         akce_ukoncit),
}

def vytiskni_menu():
    print("\n=== MENU ===")
    for klic, (nazev, _) in MAIN_DISPATCH.items():
        print(f"{klic}) {nazev}")

# ---------- Hlavní smyčka ----------

🔚 Shrnutí: co teď umíš
  • Definovat a volat funkce; rozlišit definici × volání.
  • Pracovat s parametry, argumenty, pojmenovanými a výchozími argumenty.
  • Volit return vs. print, vracet n-tice a rozbalovat je.
  • Vnímat funkce jako hodnoty a ukládat je do struktur (klíč pro dispatcher).
  • Bezpečně číst čísla (try/except + while True + validace).
  • Pracovat se seznamem n-tic, enumerate a výpočty součtů.
  • Postavit dispatcher a řídit menu programu.
Definice × volání (na prstech)
Definice = vznikne „krabička“ s kódem (nic se nespustí). Volání = krabičku skutečně použiješ. pozdrav (bez závorek) je odkaz na funkci; pozdrav() ji spouští.
Parametry vs. argumenty
def plus(a, b):  # a, b = parametry
    return a + b

plus(2, 3)       # 2, 3 = argumenty
Parametry = „škatulky“, argumenty = „co do škatulek vložím při volání“.
Proč ne měnitelné výchozí argumenty?
Výchozí hodnota se vyhodnotí při definici funkce. Měnitelné typy (list, dict) by se sdílely mezi voláními. 
def pridej(x, cil=None):
    if cil is None:
        cil = []
    cil.append(x)
    return cil
Proč raději return než print u výpočtů?
  • return vrací data – můžu je uložit, skládat, testovat.
  • print jen zobrazí – data dál nemám.
Rozbalování n-tic (unpacking)
Když funkce vrátí n-tici, lze ji rozdělit: a, b = vrat_dvojici(). Počet položek se musí shodovat s počtem proměnných.
Jak funguje try/except ValueError?
V try zkusíš převod vstupu. Když selže, vyhodí se ValueError a běh skočí do except. Smyčka while True zajistí opětovný dotaz.
Co přesně dělá while v tomhle vzoru?
Smyčka běží, dokud nedojde k validnímu vstupu a return funkci neukončí.
Proč enumerate?
Číslování položek bez ručního čítače. start=1 je přívětivé pro uživatele.
Proč dispatcher místo if/elif?
Rozšiřitelnost: přidat akci = přidat položku do slovníku. Kód zůstává krátký a přehledný.
Jak funguje dict.get(klíč, výchozí)?
Vrátí hodnotu pro klíč nebo výchozí, pokud klíč neexistuje – bez výjimky KeyError.
Funkce jako hodnoty (first-class)
Funkci můžeš uložit do proměnné, seznamu, slovníku; můžeš ji předat i vracet. Díky tomu skládáme menu a pluginy z malých kousků kódu.
