3.1.7 (1/6) Rozměry prvků: width, height, px vs %

Blokové prvky (např. <div>) bez stylů vyplní šířku řádku; výška se přizpůsobí obsahu. Rozměry řídí width (šířka) a height (výška). Jednotky: px (pevné) a % (relativní k rodiči).

Praktická cvičení (HTML → CSS)

Do index.html vlož část „výstupu studentů“:

<section class="demo-wrap">

  <h3>1) Pevný box v px</h3>
  <div class="example-1">400 × 400 px</div>

</section>

Do styles.css přidej základ a box:

body { 
  margin: 0; 
  padding: 0; 
  font-family: system-ui, sans-serif;
}

.demo-wrap { padding: 16px; }

/* 1) Pevná velikost */
.example-1 {
  width: 400px;
  height: 400px;
  background: steelblue;
  color: white;
  display: grid; 
  place-items: center;    /* zarovnání textu doprostřed (X i Y) */
  margin-bottom: 16px;
}

Co dělá display:grid; place-items:center;?

Do index.html dál přidej:

<h3>2) Dítě s výškou 50 %</h3>
<div class="parent">
  <div class="child">50 % = 100 px</div>
</div>
Vysvětlení: Pro height: 50% u dítěte musí mít rodič definovanou výšku (např. 200 px).

Do styles.css přidej:

/* 2) Parent s pevnou výškou, dítě s výškou v % */
.parent {
  width: 50%;
  height: 200px;          /* pevná výška → dítě může mít % výšku */
  background: lightgreen;
  margin-bottom: 16px;
}

/* Child s výškou 50 % → = 100 px */
.child {
  width: 150px;           /* pevná šířka dítěte */
  height: 50%;            /* 50 % z výšky parenta = 100 px */
  background: lightskyblue;
  display: grid; 
  place-items: center;
}

Proč je výška v % závislá na rodiči?

Do index.html přidej třetí blok:

<h3>3) Box uvnitř boxu: 100 % </h3>
<div class="example-2">
  <div class="example-3">100 %</div>
</div>
Idea: .example-3 má vyplnit rodiče .example-2 na 100 % šířky i výšky.

Do styles.css přidej:

/* 3) Rodič pro 100 % */
.example-2 {
  width: 400px;
  height: 400px;
  background: #447aa4;
  color: white;
}

/* Dítě: vyplň rodiče na 100 % */
.example-3 {
  width: 100%;
  height: 100%;
  background: lightgreen;
  display: grid; 
  place-items: center;
}

💡 100 % šířka/výška dítěte se odvíjí od aktuálního content boxu rodiče.

Do index.html přidej dvě ukázky gridu (jen pro zarovnání textu):

<h3>Grid systém</h3>
<div class="grid-example1">Y nahoře, X uprostřed</div>
<div class="grid-example2">Y dole, X uprostřed</div>

Grid tu nepoužíváme pro layout stránky, ale jen pro pohodlné zarovnání obsahu.

Do styles.css přidej:

/* Grid systém – jen pro zarovnání obsahu (ne layout stránky) */
.grid-example1 {
  width: 400px;
  height: 400px;
  background: #b4dc3c;
  display: grid;
  place-items: start center;  /* Y nahoru, X doprostřed */
  margin-bottom: 16px;
}

.grid-example2 {
  width: 400px;
  height: 400px;
  background: #b4dc3c;
  display: grid;
  place-items: end center;    /* Y dolů, X doprostřed */
  margin-bottom: 16px;
}

Více o place-items v modálu

  • Protože rodič nemá výšku. Procentuální výška musí z něčeho počítat.
  • Řešení: nastav rodiči height (px) nebo html, body { height: 100%; } a řetěz procent navrch.
📘 Shrnutí (část 1)
  • width/height: px = pevné, % = relativní k rodiči.
  • U height:% musí mít parent výšku.
  • Vyzkoušeno: pevný box 400×400 px, parent (50 % šířky) s child (50 % výšky), a dítě vyplňující rodiče na 100 %.
  • display:grid; place-items: … používáme jen pro zarovnání obsahu.
Jednotky v CSS
  • px – pevná velikost (stabilní, nereaguje na šířku okna).
  • % – relativní k rodiči (u height:% musí mít rodič výšku).

Příklad: .box { width: 400px; height: 400px; } → 400×400 px. .box { width: 50%; } → polovina šířky rodiče.

Vztah rodič–dítě (Parent–Child)
<div class="parent">
  <div class="child">Dítě</div>
</div>
  • % se vztahují k rozměrům parenta.
  • U height:% musí být výška parenta známa (px, rem, nebo řetězec % od kořene).

Pozn.: Šířka v % se obvykle odvíjí od šířky parenta automaticky (proto 50 % funguje hned), ale výška nikoli.

🎯 Grid: rychlé zarovnání obsahu

display: grid; + place-items slouží k pohodlnému zarovnání vnitřku prvku (ne k rozvržení celé stránky):

/* X i Y doprostřed */
.box { display: grid; place-items: center; }

/* Y nahoru, X doprostřed */
.box { display: grid; place-items: start center; }

/* Y dolů, X doprostřed */
.box { display: grid; place-items: end center; }

Alternativa: display:flex; align-items:center; justify-content:center;

3.1.7 (2/6) Rámečky – tloušťka, styl, barva + různé strany

Rámeček tvoří trojice vlastností: border-width, border-style, border-color. Bez border-style se rámeček nezobrazí. Existují i zkratky (např. border: 5px solid black;) a varianty pro jednotlivé strany (border-top, border-right…).

Rámeček v CSS – co potřebuje
Základní trojice
.box {
  border-width: 10px;      /* tloušťka */
  border-style: solid;     /* solid | dashed | dotted | double | groove | ridge | inset | outset | none */
  border-color: tomato;    /* barva tenké linie */
}

Bez border-style je rámeček „neviditelný“ (i když má width a color).

Zkratka (shorthand)
.card { border: 5px solid black; }

Pořadí: tloušťka → styl → barva.

Jednotlivé strany
.box {
  border-top:    2px dotted gray;
  border-right:  4px solid cornflowerblue;
  border-bottom: 6px double black;
  border-left:   3px dashed tomato;
}

Styl každé strany může být jiný (a i barva může být jiná).

Praktická cvičení (krok za krokem)

Do index.html (do <body>) přidej:

<section class="demo-wrap">
  <h4>Rámečky (border)</h4>

  <div class="panel">Základní border (width + style + color)</div>

  <div class="weird">Různé styly na každé straně</div>

  <div class="card">Zkratka border</div>
</section>

Do style.css přidej (hodnoty vycházejí z výuky):

/* doporučený základ z minula */
body { margin: 0; padding: 0; font-family: system-ui, sans-serif; }
.demo-wrap { padding: 16px; }

/* 400 × 400 – div bude dobře vidět */
.panel {
  width: 400px;
  height: 400px;
  background: lightgray;

  border-width: 10px;    /* tloušťka */
  border-style: solid;   /* nutné – jinak nic neuvidíš */
  border-color: tomato;  /* barva */
  display: grid; place-items: center;
  margin-bottom: 16px;
}

Ulož a zkontroluj v prohlížeči: zobrazí se šedý čtverec s červeným (tomato) rámečkem tloušťky 10 px.

Teď nastav různé styly (a barvu pravé strany) pro čtverec 300×300:

.weird {
  width: 300px;
  height: 300px;
  background: chocolate;
  border-width: 10px;   /* tloušťka všech stran */

  border-left-style:   dashed;   /* levá: čárkovaná */
  border-top-style:    dotted;   /* horní: tečkovaná */
  border-right-style:  solid;    /* pravá: plná */
  border-bottom-style: double;   /* dolní: dvojitá */
  border-right-color:  cornflowerblue;  /* jiná barva pro pravou stranu */

  display: grid; place-items: center;
  margin-bottom: 16px;
}

Výsledek: čtyři různé styly na jedné „krabici“, pravá strana navíc jinou barvou.

Jedním řádkem nastavíš šířku, styl i barvu:

.card {
  width: 260px;
  height: 120px;
  background: #fff;
  border: 5px solid black;  /* tloušťka | styl | barva */
  display: grid; place-items: center;
}

Stejně fungují i strany: border-top: 2px dashed red;

  1. Vyhledej v dokumentaci CSS vlastnosti pro border-style.
  2. Zkus groove a ridge na různých stranách a sleduj, jak se mění vzhled.
  3. Zkus změnit jen jednu stranu přes border-*-style a border-*-color.

Cílem je zvyknout si hledat přesné názvy a chování v dokumentaci a umět je použít.

📘 Shrnutí (část 2)
  • Rámeček potřebuje: border-width + border-style + border-color (bez stylu není vidět).
  • Jednotlivé strany můžeš stylovat samostatně, včetně jiné barvy.
  • Zkratka: border: <width> <style> <color>;

3.1.7 (3/6) Margin a Padding – vnější a vnitřní mezery

Margin určuje mezeru venku (mezi prvky). Padding určuje mezeru uvnitř (mezi obsahem a rámečkem). Díky nim vytváříme rozestupy mezi boxy a prostor pro text.

Margin vs. Padding – vysvětlení

Všechny prvky v CSS jsou jako krabice. Mají 4 vrstvy:

  1. Content – text, obrázky apod.
  2. Padding – prostor uvnitř kolem obsahu.
  3. Border – čára kolem boxu.
  4. Margin – prostor mezi boxy (venku).

Představ si hrnek na stole:

  • Prostor uvnitř hrnku = padding.
  • Prostor mezi hrnkem a ostatními věcmi na stole = margin.

Padding zvětšuje vnitřní prostor boxu (obsah se odtlačí od okraje). Margin posouvá celý box dál od okolních prvků.

🎯 Co dělá display: grid; place-items: center;

Tato kombinace vlastností se často používá v ukázkách, aby byl text nebo prvek přesně uprostřed boxu.

  • display: grid; – změní chování prvku, takže se jeho obsah zarovnává pomocí gridového systému (mřížky).
  • place-items: center; – zkrácený zápis pro align-items: center; (vertikálně) a justify-items: center; (horizontálně).

Výsledek: text nebo vnořený prvek se umístí přesně doprostřed – bez nutnosti řešit padding, line-height nebo flexbox.

💡 Pokud tě zajímá alternativa:
display: flex; align-items: center; justify-content: center; dělá úplně totéž, ale pomocí flexboxu.

Praktická cvičení (krok za krokem)

Do index.html vlož:

<section class="demo-wrap">
  <h4>Margin a Padding</h4>

  <div class="box">První box</div>
  <div class="box">Druhý box</div>
  <div class="box">Třetí box</div>
</section>

Všechny tři boxy budou sdílet stejnou třídu .box.

Do style.css přidej:

body { margin: 0; padding: 0; font-family: system-ui, sans-serif; }
.demo-wrap { padding: 16px; }

.box {
  background: lightgreen;
  height: 50px;
  display: grid; place-items: center;  /* zarovnání textu doprostřed */
}

Uvidíš tři zelené boxy přilepené na sebe. Zatím žádný prostor mezi nimi.

❓ Klikni sem a zjisti, co dělá display: grid; place-items: center;

Uprav CSS a přidej margin 20 px kolem každého boxu:

.box {
  background: lightgreen;
  height: 50px;
  margin: 20px;      /* vnější mezera kolem */
  display: grid; place-items: center;
}

Teď mezi boxy uvidíš prostor. Každý má kolem sebe 20 px volného místa.

Uprav .box a přidej padding: 16px; – text se posune od okraje dovnitř:

.box {
  background: lightgreen;
  padding: 16px;     /* vnitřní mezera (text není nalepený na okraji) */
  margin: 20px;      /* vnější mezera mezi boxy */
}

Všimni si, že boxy jsou teď o něco větší – padding přidává místo uvnitř, takže rozšiřuje element.

V prohlížeči klikni pravým tlačítkem → Inspect (Zkontrolovat prvek).

Najdeš tam, že i samotné <body> má výchozí margin: 8px; – přidává to prohlížeč sám.

Řešení: resetuj margin a padding u html a body na nulu:

html, body {
  margin: 0;
  padding: 0;
}

Teď stránka opravdu začíná úplně od okraje okna.

📘 Shrnutí (část 3)
  • Margin = prostor kolem prvku (vnější).
  • Padding = prostor uvnitř prvku (vnitřní).
  • Výchozí margin má i samotné body – resetuj na 0 px.
  • display: grid; place-items: center; zarovná text doprostřed boxu (horizontálně i vertikálně).

3.1.7 (4/6) Box model – jak se počítá výsledná velikost prvku

Každý HTML prvek je „krabice“ (box) složená z vrstev: content → padding → border → margin. Šířka/výška, které v CSS nastavíš, se standardně týkají jen obsahu (content). Padding a border se k rozměrům přičítají.

Box model – vrstvy a výpočty
  • Content – skutečný obsah (text, obrázky). width/height se ve výchozím režimu vztahují sem.
  • Padding – vnitřní odsazení, „tlačí“ obsah od okraje dovnitř. Zvětšuje celkový rozměr prvku.
  • Border – rámeček okolo prvku. Zvětšuje celkový rozměr prvku.
  • Margin – vnější mezera kolem prvku (nepočítá se do velikosti prvku, ale posouvá ho v layoutu).

Výpočet (výchozí režim content-box):

  • Celková šířka = content width + padding-left + padding-right + border-left + border-right
  • Celková výška = content height + padding-top + padding-bottom + border-top + border-bottom

O tom, jak to změnit na „pevnější“ chování pomocí box-sizing: border-box;, bude následující část.

Kontrola v DevTools (Box Model panel)
  1. Otevři stránku v Chromu/Edge → stiskni F12 (nebo pravým → Zkontrolovat).
  2. V panelu Elements vyber svůj prvek (např. .demo).
  3. Vpravo v části Computed najdi schéma „Box model“ – uvidíš content, padding, border a margin s konkrétními čísly.

Barevné zvýraznění při najetí myší ti ukáže přesně, která vrstva zabírá jaké místo.

Praktická ukázka a počítání (krok za krokem)

Do index.html (do <body>) vlož:

<section class="demo-wrap">
  <h4>Box model – content, padding, border, margin</h4>

  <div class="demo">Obsah (content)</div>
</section>

Budeme s tímto boxem postupně pracovat (přidáme různé paddingy a border) a budeme počítat rozměry.

Do style.css přidej (hodnoty přesně takto, ať se shodneme na číslech):

/* základ (přehlednost a font) */
body { margin: 0; padding: 0; font-family: system-ui, sans-serif; }
.demo-wrap { padding: 16px; }

/* DEMO BOX – content má 300 × 50 px */
.demo {
  width: 300px;           /* content width */
  height: 50px;           /* content height */
  background-color: lightcoral;
  margin: 20px;           /* vnější mezera – nepočítá se do velikosti boxu */
  /* asymetrický padding (všimni si různých směrů): */
  padding-top: 100px;
  padding-right: 90px;
  padding-bottom: 50px;
  padding-left: 120px;

  /* jen pro čitelnost textu uprostřed */
  display: grid; place-items: center;
}

Teď uvidíš „tlustý“ červený box – padding ho uvnitř roztlačí od obsahu a celkově se zvětší.

🎯 Co dělá display: grid; place-items: center;?

Otevři DevTools → vyber .demo → panel Computed → schéma Box model.

Výpočet rozměrů (bez borderu)

Zadané hodnoty: content = 300×50, padding = 100 (top), 90 (right), 50 (bottom), 120 (left)

  • Celková šířka = 300 + 120 + 90 = 510 px
  • Celková výška = 50 + 100 + 50 = 200 px

Margin (20 px) je vnější – nepřičítá se do velikosti prvku, jen odtlačí prvek od okolí.

Do třídy .demo přidej border (na všechny strany):

.demo {
  /* ... vše z dřívějška ... */
  border: 10px solid black;
}

V DevTools si ověř, že se celková velikost znovu zvětšila (protože border se přičítá).

Výpočet rozměrů (s borderem)

Navíc: border = 10 px na všech stranách.

  • Celková šířka = 510 + 10 + 10 = 530 px
  • Celková výška = 200 + 10 + 10 = 220 px
Takto to funguje ve výchozím režimu content-box. V další části si ukážeme, jak box-sizing: border-box; „vtlačí“ padding a border dovnitř, aby celková šířka/výška zůstala přesně na zadané hodnotě.

  1. Změň padding-right na 110 px a padding-bottom na 70 px.
  2. Ručně přepočti celkovou šířku/výšku a ověř v DevTools (Computed → Box model).
  3. Přidej border-left: 20px solid #333; a znovu uprav výpočet (pozor – různé tloušťky na stranách).
📘 Shrnutí (část 4)
  • Prvek = content + padding + border (+ margin mimo prvek).
  • Ve výchozím režimu content-box se padding a border k width/height přičítají → prvek se zvětšuje.
  • Výpočty si vždy ověř v DevTools → Computed → Box model.
  • V další části ukážeme box-sizing: border-box;, který udrží celkovou velikost prvku stabilní.
🎯 Co dělá display: grid; place-items: center;

Snadný způsob, jak zarovnat obsah prvku doprostřed (horizontálně i vertikálně):

  • display: grid; aktivuje mřížku pro vnitřní zarovnání.
  • place-items: center; = align-items: center; + justify-items: center;.

Alternativa: display:flex; align-items:center; justify-content:center;

3.1.7 (5/6) Box-sizing – jak se šířka a výška opravdu počítá

V předchozí části jsme viděli, že prvek se ve výchozím nastavení zvětšuje, protože padding a border se přičítají k šířce a výšce. Vlastnost box-sizing nám umožní to změnit – říct prohlížeči, co přesně má být započteno do rozměru prvku.

Jak funguje box-sizing

Vlastnost box-sizing říká, jak se má počítat šířka a výška prvku. Může mít dvě hlavní hodnoty:

  • content-box (výchozí) – šířka/výška se vztahuje pouze na obsah.
  • border-box – šířka/výška zahrnuje i padding a border.

Příklad (šířka 200 px, padding 20 px, border 10 px):

  • content-box → celková šířka = 200 + 20 + 20 + 10 + 10 = 260 px
  • border-box → celková šířka = 200 px (padding a border jsou uvnitř)

Proto border-box dělá layout stabilnější a přehlednější – prvek nikdy nepřeroste rodiče.

Výpočet s border-box

Máme:

width: 200px;
padding: 20px;
border: 10px solid steelblue;

content-box: → obsah = 200 px, prvek = 260 px široký.

border-box: → prvek zůstává 200 px široký, ale obsah se zmenší na 140 px (200 − 20 − 20 − 10 − 10).

💡 Padding a border se vtlačí „dovnitř“ prvku, takže celková velikost zůstane stabilní.

Praktická ukázka – dva boxy se stejnými rozměry

Do index.html vlož:

<section class="demo-wrap">
  <h4>Box-sizing – content-box vs border-box</h4>

  <div class="content-box">content-box</div>
  <div class="border-box">border-box</div>
</section>

Oba prvky mají stejný obsah, rozdíl bude jen ve výpočtu rozměru.

Do style.css přidej:

.content-box, .border-box {
  width: 200px;
  height: 50px;
  background: lightblue;
  margin: 8px 0;
  padding: 20px;
  border: 10px solid steelblue;
  display: grid; place-items: center;
}

/* rozdíl v režimu výpočtu */
.content-box { box-sizing: content-box; }  /* výchozí chování */
.border-box  { box-sizing: border-box;  }  /* stabilní velikost */

Oba boxy mají stejný kód – liší se pouze v režimu box-sizing. Sleduj, že content-box se zvětší, zatímco border-box zůstává přesně 200 px široký.

Otevři F12 → klikni na každý box → panel Computed → Box model.

Porovnej:

  • content-box: celková šířka ≈ 260 px (200 + padding + border)
  • border-box: celková šířka = 200 px (padding a border „uvnitř“)

👉 border-box se hodí hlavně při návrhu layoutů s přesnými rozměry (karty, gridy, inputy).

V reálných projektech se často používá tzv. reset pro všechny prvky:

* {
  box-sizing: border-box;
}

Tím zajistíš, že šířky a výšky všech prvků se budou počítat konzistentně.

✅ Doporučení: použij tento reset hned na začátku každého projektu – ušetří ti desítky minut ladění.
📘 Shrnutí (část 5)
  • content-box = výchozí režim, padding a border zvětšují prvek.
  • border-box = šířka/výška zahrnuje i padding a border → stabilní velikost.
  • Celková šířka v content-box: width + padding + border.
  • Doporučený reset: * { box-sizing: border-box; }.
  • V DevTools sleduj rozdíl – border-box „držící tvar“ je tmavší a přesně ohraničený.

3.1.7 (6/6) Univerzální selektor * a výchozí styly prohlížeče

Každý prohlížeč přidává HTML prvkům výchozí styly – mezery, paddingy nebo písma. Proto se ti někdy zdá, že má stránka „okraj“, i když jsi žádný nenastavil. Univerzální selektor (*) umožňuje upravit všechny prvky najednou a začít s čistým plátnem.

Co je univerzální selektor *

Selektor * znamená „vyber všechny prvky“. Používá se například pro nastavení jednotného chování celého webu: 

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Tento zápis odstraní výchozí mezery u všech prvků a zajistí, že jejich rozměry se počítají stabilně.

💡 Tento zápis se nazývá CSS reset nebo normalize. Je vhodné ho mít hned na začátku každého projektu.
Výchozí styly prohlížeče

Každý prohlížeč (Chrome, Firefox, Safari…) má vlastní uživatelský CSS soubor, kde jsou nastavené výchozí styly.

  • <body> má obvykle margin: 8px;
  • <ul> a <ol> mají padding-left: 40px;
  • <h1> až <h6> mají předdefinované velikosti a mezery.
  • <p> má výchozí margin-top a margin-bottom.

Když tyto hodnoty vynuluješ pomocí univerzálního selektoru, můžeš si rozvržení vytvořit zcela podle sebe.

⚠️ Pozor – resetuj styly jen na začátku projektu. U hotových šablon může univerzální selektor rozbít hotový layout.

Praktické cvičení – resetujeme výchozí styly

Do index.html vlož následující kód:

<h1>Nadpis první úrovně</h1>
<p>Toto je odstavec s výchozím stylem prohlížeče.</p>

<ul>
  <li>Položka 1</li>
  <li>Položka 2</li>
</ul>

Když stránku otevřeš, uvidíš mezery a odsazení, i když jsi žádné nestanovil.

V DevTools si všimni:

  • <body> má margin: 8px;
  • <ul> má padding-left: 40px;
  • <h1> a <p> mají své vlastní mezery.

Toto vše nastavuje prohlížeč automaticky.

Do style.css na úplný začátek napiš:

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Nyní stránka ztratí všechny výchozí mezery. Nadpis, odstavec i seznam se zarovnají přesně k okraji.

✅ Výsledek: „čisté plátno“ – odteď máš rozložení zcela pod kontrolou.

Po resetu si mezery nastavíš ručně podle potřeby:

body { padding: 20px; font-family: system-ui, sans-serif; }
h1   { margin-bottom: 12px; }
p    { margin-bottom: 8px; }
ul   { margin-left: 20px; }

Tím máš kontrolu nad celým vzhledem stránky bez vlivu výchozích hodnot prohlížeče.

Cvičení 3.2.3 – Rozšiř kartu na mini portfolio

Z jednoduché osobní karty teď vytvoříš přehlednější mini portfolio panel. V této části si procvičíš hlavně rozměry, rámečky, margin, padding, box model a rozdíl mezi content-box a border-box.

Tvůj cíl

Udělat z jedné karty větší panel, který bude rozdělený na několik sekcí.

Kroky
  1. Navazuj na stejný projekt jako v předchozích kapitolách.
  2. Rozhodni se, jak široké má tvoje portfolio být – nemělo by se táhnout přes celou obrazovku.
  3. Vytvoř hlavní kontejner, který bude představovat celé portfolio.
  4. Uvnitř vytvoř alespoň tři menší části, například: „O mně“, „Dovednosti“, „Kontakt“.
  5. Každé části nastav vlastní prostor uvnitř, aby text nebyl nalepený na okraje.
  6. Mezi jednotlivými částmi vytvoř rozestupy.
  7. Přidej rámeček nebo zvýraznění, aby byly bloky jasně oddělené.
  8. U jednoho boxu si vyzkoušej běžné chování výpočtu velikosti.
  9. U druhého boxu vyzkoušej border-box a porovnej rozdíl.
  10. Zkus si v DevTools zkontrolovat, jak se do velikosti prvku započítává obsah, padding a border.
  11. Na začátek stylů přidej jednoduchý reset, aby tě nerušily výchozí mezery prohlížeče.
Na co si dát pozor
  • Rozměry mají sloužit přehlednosti, ne jen mechanickému nastavování čísel.
  • Padding je vnitřní prostor, margin je vnější prostor mezi boxy.
  • Rámeček bez stylu není vidět.
  • U procentuálních rozměrů přemýšlej, vůči čemu se vlastně počítají.
Hotovo budeš mít tehdy, když:
  • tvoje osobní karta už působí jako skutečný panel mini portfolia,
  • máš na stránce několik oddělených bloků,
  • vědomě používáš šířku, padding, margin, border a box-sizing.

  • Po resetu si mezery a styly nastavujeme ručně podle designu.
