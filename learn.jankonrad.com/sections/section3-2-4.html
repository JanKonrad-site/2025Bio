3.1.8 Display, border-radius, text, fonty a centrování

Co se naučíš?

  • Rozlišit block a inline prvky.
  • Používat display a chápat block, inline, inline-block.
  • Vytvářet zaoblené rohy pomocí border-radius.
  • Stylovat text přes text-align, text-decoration, text-transform.
  • Pracovat s fonty: font-size, font-family, font-weight.
  • Vycentrovat prvek horizontálně pomocí margin: 0 auto.
  • Vycentrovat obsah vertikálně pomocí align-content: center (na rodiči).

1️⃣ Block vs Inline prvky

HTML prvky se chovají dvěma základními způsoby: block a inline. To určuje, jestli prvek začne na novém řádku a kolik zabere místa.

Block vs Inline – co to znamená
Block (blokové prvky)
  • Zabírají celou dostupnou šířku rodiče (typicky celý řádek).
  • Vždy začínají na novém řádku.
  • Příklady: h1–h6, p, div, ul, li.
Inline (řádkové prvky)
  • Nezačínají na novém řádku.
  • Zaberou jen tolik místa, kolik potřebují (jako text).
  • Příklady: a, span, strong, button, img.
Inline prvky se „chovají jako text“ – proto se řadí vedle sebe.

Do index.html vlož do <body>:

<section class="demo-wrap">
  <h3>Block prvky (začnou na novém řádku)</h3>

  <h2>Nadpis H2 #1</h2>
  <h2>Nadpis H2 #2</h2>
  <h2>Nadpis H2 #3</h2>

  <div>DIV je také block prvek</div>
  <p>Odstavec P je také block prvek</p>

  <ul>
    <li>LI je block prvek</li>
    <li>LI je block prvek</li>
  </ul>

  <hr>

  <h3>Inline prvky (řadí se vedle sebe)</h3>
  <a href="#">Odkaz 1</a>
  <a href="#">Odkaz 2</a>
  <a href="#">Odkaz 3</a>

  <button>Tlačítko</button>
  <span>Span</span>
  <strong>Strong</strong>
</section>

Do styles.css přidej:

body { margin: 0; padding: 0; font-family: system-ui, sans-serif; }
.demo-wrap { padding: 16px; }

h2 { background: #dbeafe; }          /* uvidíš, že vyplní šířku řádku */
a, button, span, strong { background: #dcfce7; }
Sleduj: H2 má pozadí přes celý řádek (block). Odkazy a span mají pozadí jen kolem textu (inline).

2️⃣ Display: změna chování prvku

Vlastnost display umí přepnout prvek mezi block, inline a inline-block. Díky tomu můžeme inline prvku dát výšku/šířku a svislé marginy.

Display: block / inline / inline-block

Inline prvky tečou v řádku jako text. Proto běžně:

  • ignorují width a height
  • ignorují svislé okraje margin-top a margin-bottom
  • vodorovné okraje (margin-left, margin-right) fungují

inline-block je kompromis: prvek je v řádku (vedle dalších), ale respektuje width, height i svislé marginy.

Prakticky: když chceš „inline prvek, který jde velikostně stylovat“, často použiješ display: inline-block. 

<section class="demo-wrap">
  <h3>Display – pokus se o margin/width u span</h3>

  <div class="d1">DIV #1</div>
  <div class="d1">DIV #2</div>

  <span class="s1">SPAN #1</span>
  <span class="s1">SPAN #2</span>
</section>

.d1, .s1 {
  border: 2px solid #111827;
  margin: 12px;
  padding: 10px;
  background: #fef3c7;
}

/* zkusíme „násilím“ span zvětšit */
.s1 {
  width: 180px;
  height: 80px;
  margin-top: 80px;   /* u inline prvku neuvidíš efekt */
}
Sleduj: U DIV se projeví margin-top a velikost. U SPAN typicky ne.

Teď postupně zkoušej tyto varianty (vždy jednu, ostatní zakomentuj):

A) SPAN jako block – začne na novém řádku a začne respektovat width/height/margins. 
/* A) span začne fungovat jako block */
.s1 { display: block; }
B) DIV jako inline – začne se chovat jako text. 
/* B) div se začne chovat jako inline */
.d1 { display: inline; }
C) inline-block – zůstane v řádku, ale jde velikostně stylovat. 
/* C) kompromis */
.s1 { display: inline-block; }

3️⃣ Rounded corners: border-radius

Zaoblené rohy dělá border-radius. Čím vyšší hodnota, tím kulatější rohy. Když je border-radius 50 %, z čtverce vznikne kruh.

border-radius – jak funguje
  • border-radius: 20px; zaoblí všechny rohy stejně.
  • Čím větší číslo, tím víc kulaté rohy.
  • border-radius: 50%; udělá kruh, pokud je width = height.
  • Rohy lze nastavit zvlášť: border-top-left-radius apod.
  • Shorthand pro 4 rohy jde po směru hodin: TL → TR → BR → BL.
/* 4 hodnoty: TL, TR, BR, BL */
.box { border-radius: 90px 60px 30px 5%; }

<section class="demo-wrap">
  <h3>border-radius demo</h3>
  <div class="radius-box">200 × 200</div>
</section>

.radius-box{
  width: 200px;
  height: 200px;
  background: steelblue;
  color: white;
  display: grid;
  place-items: center;
  border-radius: 20px;   /* zkus 5px, 50px, 100px, 50% */
}
Úkol: změň hodnotu border-radius a sleduj, kdy se z čtverce stane kruh.

4️⃣ Text: text-align, text-decoration, text-transform

Text se dá zarovnat (text-align), dekorovat čarou (text-decoration) a převést na velká/malá písmena (text-transform).

Text properties: align / decoration / transform
text-align
  • left (výchozí), center, right, justify
  • justify dorovnává řádky na stejnou délku (jako noviny)
text-decoration
  • čára: underline, overline, line-through
  • jde nastavit i barva, styl a tloušťka (shorthand, pořadí nevadí)
  • text-decoration: none; často používáme na odkazy, abychom odstranili podtržení
text-transform
  • uppercase, lowercase, capitalize
  • užitečné, když text přichází z databáze/API a nemáš kontrolu nad formátem

<section class="demo-wrap">
  <h4 class="left">Text vlevo</h4>
  <h4 class="center">Text na střed</h4>
  <h4 class="right">Text vpravo</h4>

  <p class="justify">
    Tohle je delší text, na kterém uvidíš rozdíl mezi běžným zarovnáním a justify.
    Přidej pár vět, ať je to na více řádků.
  </p>

  <a href="#" class="link">Odkaz (bez podtržení)</a>
</section>

.left   { text-align: left; }
.center { text-align: center; }
.right  { text-align: right; }

.justify { text-align: justify; }

.center{
  text-decoration: underline;
}

/* shorthand: line + color + style + thickness */
.right{
  text-decoration: underline #2563eb wavy 4px;
}

.link{
  text-decoration: none;     /* odstraní výchozí podtržení odkazu */
}

.justify{
  text-transform: capitalize;
}

5️⃣ Fonty: font-size (px/pt/em/rem) a proč je rem praktické

Pro fonty jsou klíčové: font-size, font-family, font-style, font-weight. U velikosti se často používá rem, protože měníš vše z jednoho místa (root).

Velikost fontu: px / pt / em / rem
  • px – absolutní jednotka (pevná velikost).
  • pt – body (znáš z Wordu; výchozí velikost bývá 12 pt ≈ 16 px).
  • em – násobek velikosti písma rodiče (1em = 100 % parent font-size).
  • rem – násobek velikosti písma kořene (html) → „řídím celý web z jednoho místa“.
html { font-size: 12pt; }    /* root */

h1 { font-size: 3rem; }      /* 3 × root */
p  { font-size: 1rem; }      /* 1 × root */
Když změníš html { font-size: ... }, změní se proporčně vše, co je v rem. 

<section class="demo-wrap">
  <h1>H1 – 3rem</h1>
  <h2>H2 – 2rem</h2>
  <h3>H3 – 1.5rem</h3>
  <p>Odstavec – 1rem</p>
</section>

html { font-size: 12pt; }   /* výchozí logika z transkriptu */

h1 { font-size: 3rem; }
h2 { font-size: 2rem; }
h3 { font-size: 1.5rem; }
p  { font-size: 1rem; }
Úkol: změň html { font-size: 18pt; } a sleduj, že se zvětší vše proporčně.

6️⃣ Font-family a font-weight (včetně fallbacků)

font-family vybírá font. Vždy dávej fallback (záložní fonty). font-weight určuje „tučnost“ (normal/bold nebo 100–900).

font-family: výběr fontu + fallback

Font může být na počítači nedostupný, proto se píše seznam náhrad:

p { 
  font-family: "Segoe UI", Arial, Verdana, sans-serif;
}
  • Pokud není první font, použije se další.
  • Poslední bývá obecný typ, typicky sans-serif.
font-weight: tučnost
  • normal ≈ 400
  • bold ≈ 700
  • Čísla: 100–900 (čím vyšší, tím tučnější) – záleží na tom, co font podporuje.
  • lighter a bolder jsou relativní k rodiči (v transkriptu: ±300).
p { font-weight: 700; }    /* bold */
h1{ font-weight: normal; } 

<section class="demo-wrap">
  <h3 class="fw-normal">Nadpis (normal)</h3>
  <p class="fw-bold">Odstavec (bold)</p>
  <p class="fw-100">Odstavec (100)</p>
  <p class="fw-700">Odstavec (700)</p>
</section>

body{
  font-family: "Segoe UI", Arial, Verdana, sans-serif;
}

.fw-normal { font-weight: normal; }  /* 400 */
.fw-bold   { font-weight: bold; }    /* 700 */

.fw-100    { font-weight: 100; }
.fw-700    { font-weight: 700; }
Pozor: některé fonty neumí všechny váhy (100–900). Výsledek záleží na fontu.

7️⃣ Centrovaní: margin: 0 auto + align-content: center

Horizontálně vycentruješ blokový prvek pomocí margin: 0 auto. Vertikální centrování obsahu v rodiči uděláš přes align-content: center (aplikuje se na rodiče).

Horizontální centrování: margin: 0 auto

Když nastavíš margin-left i margin-right na auto, prohlížeč rozděluje volné místo na obě strany stejně → prvek je uprostřed. 

.box{
  width: 50%;
  height: 300px;
  margin: 0 auto;   /* top/bottom 0, left/right auto */
}
Tohle funguje na blokové prvky s definovanou šířkou (ne na inline).
Vertikální centrování obsahu: align-content: center

align-content: center; dáš na rodiče, aby jeho děti byly vertikálně uprostřed.

Pozor: aby to fungovalo přes celou obrazovku, musí mít rodič skutečnou výšku. V transkriptu se to řeší html, body { height: 100%; } a odstraněním default marginu. 

html, body{
  height: 100%;
  margin: 0;
}

body{
  align-content: center;
}

<section class="demo-wrap">
  <div class="center-box">
    <p class="center-text">Text uprostřed</p>
  </div>
</section>

.center-box{
  width: 50%;
  height: 300px;
  background: steelblue;
  margin: 0 auto;        /* horizontálně uprostřed */
}

.center-text{
  width: 50%;
  height: 150px;
  background: #93c5fd;
  margin: 0 auto;        /* horizontálně uprostřed uvnitř boxu */
  text-align: center;    /* text na střed (vodorovně) */
  align-content: center; /* text vertikálně uprostřed (na rodiči prvků uvnitř) */
}
Úkol z transkriptu: vycentruj i celý box vertikálně v rámci stránky.

Přidej (nebo uprav) tyto styly přesně podle logiky z transkriptu:

html, body{
  height: 100%;
  margin: 0;        /* odstraní výchozí okraj a scrollbar */
}

body{
  align-content: center;  /* vertikálně centruje děti body */
}
Proč je potřeba height: 100% i na html?
Protože body počítá procenta z rodiče – a rodičem body je html.
Cvičení 3.2.4 – Doladění vzhledu, textu a centrování

Tvoje mini portfolio už má strukturu. Teď mu dáš příjemnější vzhled pomocí display, border-radius, práce s textem, fontů a centrování.

Tvůj cíl

Upravit portfolio tak, aby působilo moderněji, čitelněji a víc jako skutečná profilová karta.

Kroky
  1. Do horní části portfolia přidej místo pro profilovou fotografii nebo její zástupný prvek.
  2. Vyzkoušej, jak z obyčejného čtverce uděláš kruh nebo zaoblený avatar.
  3. Uprav nadpisy a text tak, aby měly jasnou hierarchii.
  4. Nastav jiné velikosti písma pro hlavní nadpis, podnadpis a běžný text.
  5. Vyzkoušej jiný font nebo bezpečný seznam náhradních fontů.
  6. Uprav zarovnání textu – některé části nech na střed, jiné třeba vlevo.
  7. Přidej odkaz nebo tlačítko, například „Kontaktuj mě“ nebo „Zjistit více“.
  8. Odstraň výchozí podtržení odkazu a uprav jeho vzhled.
  9. Vyzkoušej rozdíl mezi block, inline a inline-block na menších prvcích.
  10. Vycentruj hlavní kartu na stránce horizontálně.
Na co si dát pozor
  • Fonty a velikosti písma mají zlepšovat čitelnost, ne stránku přeplnit.
  • Zaoblení je detail, který má podpořit vzhled, ne být samoúčelný.
  • Nezaměňuj centrování textu s centrováním celého prvku.
  • U odkazu si ověř, že je stále poznat, že jde o klikací prvek.
Hotovo budeš mít tehdy, když:
  • portfolio má jasnou typografii a uhlazenější vzhled,
  • obsahuje zaoblený prvek nebo avatar,
  • obsahuje odkaz nebo tlačítko,
  • hlavní karta je přehledně umístěná a působí vyváženě.

📘 Shrnutí kapitoly

  • Block = nový řádek + plná šířka; Inline = jako text, jen potřebné místo.
  • display umí měnit chování: block, inline, inline-block.
  • border-radius dělá zaoblené rohy, 50% dělá kruh (u čtverce).
  • Text: text-align, text-decoration, text-transform.
  • Fonty: rem je praktické pro jednotné měřítko, fallbacky u font-family.
  • Centrování: margin: 0 auto pro bloky horizontálně; align-content: center na rodiči pro vertikální centrování.
