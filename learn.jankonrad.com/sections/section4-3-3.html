4.3.3. Základy OOP: Třída Hráč

Co se naučíš?

V této kapitole si poprvé vytvoříme vlastní třídu a pochopíme základní principy objektově orientovaného programování v Pythonu.

  • Třída a instance: Jak vytvořit vlastní objekt podle šablony.
  • Metoda __init__: Jak se vytváří konstruktor a ukládají vlastnosti.
  • Metody a atributy: Jak přidávat funkce do objektu a uchovávat data.
Tip: OOP je klíčovým přístupem v moderním programování – používá se třeba i při tvorbě her.
  • 1. Vytvoření třídy
    • Nejprve vytvoříme třídu pomocí klíčového slova class: 
      class Hrac:
    def __init__(self, jmeno):
        self.jmeno = jmeno
        self.zivoty = 5
        self.inventar = []
  • 2. Přidání metody seber()
    • Metoda je funkce uvnitř třídy. Přidáme například metodu pro sbírání předmětů: 
      def seber(self, predmet):
    self.inventar.append(predmet)
    print(self.jmeno, "sebral:", predmet)
  • 3. Vytvoření objektu hráče
    • Objekt vytvoříme voláním třídy jako funkce: 
      hrac = Hrac("Jan")

      Objekt hrac má nyní jméno, životy a prázdný inventář.

  • 4. Volání metod a práce s objektem
    • Nyní můžeme s objektem pracovat: 
      hrac.seber("meč")
hrac.seber("lektvar")

print(hrac.inventar)
print(hrac.jmeno)
print(hrac.zivoty)

Úkoly:

  • 1. Přidej další metodu, například vypis_statistiky().
  • 2. Umožni uživateli zadat jméno hráče pomocí input().
  • 3. Vytvoř si vlastní příběh a přidej další interakce.

Shrnutí kapitoly

V této kapitole ses dozvěděl:

  • class: jak vytvořit třídu v Pythonu.
  • __init__(): jak inicializovat objekt.
  • Metody: jak přidat chování objektu.
Tip: Tvoř si vlastní třídy, programy budou přehlednější a zábavnější.
Metoda __init__

Konstruktor __init__ se spustí automaticky při vytvoření objektu. Slouží k nastavení počátečních hodnot.

def __init__(self, jmeno):
    self.jmeno = jmeno
    self.zivoty = 5
    self.inventar = []

self odkazuje na konkrétní objekt – viz níže.

Co je metoda třídy?

Metoda je funkce uvnitř třídy, která může pracovat s atributy objektu přes self.

def seber(self, predmet):
    self.inventar.append(predmet)
    print(self.jmeno, "sebral:", predmet)

Metoda má vždy první parametr self, který odkazuje na aktuální objekt.
