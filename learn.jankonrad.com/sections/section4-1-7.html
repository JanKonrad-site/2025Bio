4.1.7 Funkce krok za krokem – od volání po načítání vstupů

V této lekci pochopíš, jak se tvoří a používají funkce v Pythonu. Každé cvičení navazuje na předchozí a připravuje tě na projekt „Kalkulačka“.

Začneme úplným základem — vytvoříme funkci a zavoláme ji:

def pozdrav():
    print("Ahoj")

pozdrav()
  • def – klíčové slovo, kterým začíná definice funkce.
  • pozdrav() – volání funkce (závorky = spuštění).
Mini-úkol: vytvoř def info(): která vypíše "Program běží" a zavolej ji dvakrát.

Chceme, aby funkce uměla počítat s hodnotami, které jí pošleme.

def soucet(a, b):
    print(a + b)

soucet(2, 3)  # vypíše 5
  • Parametry (a, b) jsou proměnné, které přijímají hodnoty.
  • Argumenty (2, 3) jsou skutečné hodnoty při volání.
Mini-úkol: napiš def pozdrav(jmeno, vek) a vypiš f"Ahoj {jmeno}, je ti {vek} let."

Někdy nechceme hned tisknout výsledek, ale vrátit ho zpět pro další výpočty.

def soucet(a, b):
    return a + b   # vrátí výsledek, netiskne

vysledek = soucet(4, 6)
print("Součet je:", vysledek)
Zapamatuj: print() zobrazuje, return vrací.
Mini-úkol: napiš def rozdil(a,b), která vrátí a-b, a výsledek ulož do proměnné.

def soucet(a,b):
    return a + b

def rozdil(a,b):
    return a - b

Soucet = soucet   # uložím odkaz (bez závorek!)
Rozdil = rozdil

print(Soucet(10,5))
print(Rozdil(10,5))

Toto budeme později využívat při tvorbě „menu“ s výběrem operací.

Mini-úkol: ulož si funkci rozdil pod jiný název a zavolej ji.

Při práci s uživatelem může dojít k chybě, např. zadá text místo čísla. Proto použijeme výjimku – záchrannou síť try / except.

def nacti_cislo():
    while True:
        try:
            return int(input("Zadejte číslo: "))
        except ValueError:
            print("Tohle není celé číslo, zkuste to znovu.")
  • try = pokus se o převod na celé číslo.
  • except ValueError = pokud to nejde, vypiš varování a opakuj.
  • return = když převod proběhne úspěšně, funkce skončí a vrátí číslo.
Tento princip použijeme v kalkulačce pro načítání čísel.
Mini-úkol: spusť funkci několikrát – zkus zadat text, pak číslo.

def nacti_dve_cisla():
    print("Načteme dvě čísla:")
    a = nacti_cislo()
    b = nacti_cislo()
    return a, b

x, y = nacti_dve_cisla()
print("Zadal jsi:", x, "a", y)
print("Součet je:", x + y)
  • Funkce nacti_dve_cisla() využívá dříve vytvořenou nacti_cislo().
  • Vrací dvě hodnoty jako dvojici (tuple), které si rozdělíme na x, y.
Tohle je ideální příprava pro kalkulačku – už umíme načíst vstupy bezpečně.
💡 Shrnutí
  • def = definice funkce.
  • print() = zobrazí, return = vrátí hodnotu.
  • Parametry = vstup do funkce, argumenty = hodnoty při volání.
  • try / except = ošetření chyb při zadávání vstupů.
  • Funkce může vracet jednu nebo více hodnot (např. a, b).
